หากเกิดปัญหาในการเรียกโปรแกรมให้แก้ไขดังนี้

  1.หากไม่สามารถเรียกตัวโปรแกรม โดยโปรแกรมจะถามหาไฟล์ link ให้เอา VC++ 
version 6.0 มาลงที่เครื่อง แล้วให้ VC++ complie อีกครั้งจะสร้าง link ให้ใหม่

  2.หากไม่สามารถให้โปรแกรมเริ่มทำการค้นหา อาจเป็นเพราะไม่มี Directory 
ที่โปรแกรมต้องการ ให้สร้าง directory "hisDB" บน subdirectory เดียวกับ
ตัวโปรแกรม ซึ่งเป็น default directory ที่โปรแกรม diblook จะเรียก
ก่อนที่จะเริ่มทำงานการค้นหาภาพ 
  อ่านการใช้งานได้ที่คู่มือการติดตั้งและการใช้งานโปรแกรม
