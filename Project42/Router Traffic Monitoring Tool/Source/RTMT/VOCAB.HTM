Vocabulary Meaning

daily graph เป็นกราฟแสดงข้อมูลรายวัน โดยแกนนอนจะเป็นหน่วยเวลาเป็นนาที

monthly graph เป็นกราฟแสดงข้อมูลรายเดือน โดยแกนนอนจะเป็นหน่วยเวลาเป็นวัน

yearly month เป็นกราฟแสดงข้อมูลรายปี โดยแกนนอนจะเป็นหน่วยเวลาเป็นลำดับของเดือน

 

 

octet เป็นตัวเลือก หมายถึง ข้อมูลเข้าออกจากเราท์เตอร์

error เป็นตัวเลือก หมายถึง ข้อมูลผิดพลาดที่รับเข้ามาและส่งออกจากเราท์เตอร์

ประเภทของกราฟข้อมูล

package เป็นกราฟของแพ็คเกจข้อมูลที่รับเข้ามาและส่งออกจากเราท์เตอร์

sum เป็นกราฟของผลรวมของแพ็คเกจข้อมูลที่รับเข้ามาและส่งออกจากเราท์เตอร์ตั้งแต่เริ่มเก็บข้อมูล

average เป็นกราฟของค่าเฉลี่ยของแพ็คเกจข้อมูลที่รับเข้ามาและส่งออกจากเราท์เตอร์ตั้งแต่เริ่มเก็บ

different เป็นกราฟของค่าผลต่างของแพ็คเกจที่รับเข้ามาและส่งออกจากเราท์เตอร์เปรียบเทียบกับค่าแพ็คเกจ ที่รับเข้ามาจากช่วงเวลาก่อนๆ
