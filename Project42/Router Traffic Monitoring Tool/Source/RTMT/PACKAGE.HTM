Package Graph Meaning

  กราฟแสดงจำนวนแพ็คเกจที่รับเข้ามา (แสดงด้วยสีเขียว)

  และจำนวนแพ็คเกจที่ส่งออกไป (แสดงด้วยสีแดง)

  ซึ่งในที่นี้เป็นกราฟแสดงผลรายวัน สังเกตได้จากในแกนนอน

 

 

  ที่หน่วยของกราฟเป็นชั่วโมง

  โดยถ้าเป็นกราฟรายเดือน แกนนอนจะมีหน่วยเป็นวันที่

  และจะมีหน่วยเป็นเลขลำดับของเดือน สำหรับกราฟรายปี

  การแปลความหมายกราฟสามารถดูได้โดยตรงจากกราฟ

  โดยแกนตั้งแสดงถึงปริมาณของแพ็คเกจเข้าและออก
