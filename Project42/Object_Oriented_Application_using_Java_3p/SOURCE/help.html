วิธีใช้งานโปรแกรม

ผู้ใช้ที่เข้ามาใช้งานเป็นครั้งแรกต้องทำการ
ลงทะเบียนก่อน โดยคลิกที่ "ลงทะเบียน"
ที่หน้าจอหลัก หลังจากนั้นให้กลับมาที่
หน้าจอหลักอีกครั้งเพื่อกรอกชื่อติดต่อบริการ
และรหัสผ่าน จากนั้นผู้ใช้สามารถใช้บริการ
รับเมล์, ส่งเมล์ และบริการฟรีโฮมเพจได้
* ในกรณีที่ใช้บริการฟรีโฮมเพจผู้ใช้ต้องตั้ง
ระดับของความปลอดภัยต่ำเพื่อให้สามารถ
