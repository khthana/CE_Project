เงื่อนไขการกำหนดชื่อติดต่อบริการ
1. ชื่อติดต่อบริการ จะต้องมีความยาว 3-30 ตัวอักษร
2. อักขระที่ใช้ต้องเป็นอักขระภาษาอังกฤษตัวเล็ก (a-z) หรือ ตัวเลข (0-9) เท่านั้น (อ่านข้อ 3 เพิ่มเติม)
3. ยอมให้ใช้อักขระพิเศษ แค่ 2 ตัวเท่านั้นคือ ขีดล่าง("_") หรือ จุด(".")
4. อักขระที่เริ่มต้น และลงท้าย ต้องเป็นอักขระภาษาอังกฤษตัวเล็ก (a-z)
ชื่อติดต่อบริการ ไม่ต้องใส่ @Pmail.com
กำหนดรหัสผ่าน ตัวอักษรภาษาอังกฤษ และตัวเลข ความยาว 4 - 30 ตัวอักษร
