เนื่องจากการอ่าน Harddisk Serial Number อาศัยฟังก์ชันภายในของ SMARTVSD.VXD แต่ใน ระบบปฏิบัติการ MS Windows 9x ไม่ได้กำหนดให้ดีไวซ์ไดร์เวอร์ดังกล่าวทำงาน การเปิดให้ดีไวซ์ไดร์เวอร์ดังกล่าวทำงานให้ทำโดยการ Copy ไฟล์ SMARTVSD.VXD ซึ่งปกติจะอยู่ในโฟลเดอร์ System ของวินโดว์ไปยังไฟลเดอร์ System\IOSUBSYS ของวินโดว์ แล้วทำการ Restart วินโดว์ใหม่ SMARTVSD.VXD ก็จะเริ่มทำงานโดยอัติโนมัติ
