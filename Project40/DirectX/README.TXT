DXSDK.ZIP    ชุดพัฒนาDirectXแบบเต็ม
INSTALL <DIR>tool และโปรแกรมที่ใช้ในการพัฒนา DirectX
REPORT  <DIR>รายงานโปรเจ็ก
DXSDK   <DIR>ชุดพัฒนาฉบับย่อมีแค่ include และ library
CDX     <DIR>Library ช่วยให้การใช้DirectX ทำได้ง่ายยิ่งขึ้น
CODE    <DIR>code ของการทำโปรเจ็กชุดนี้

หมายเหตุ การติดตั้ง DirectX SDK จำเป็นต้องใช้เนื้อที่อย่างน้อย 100MB
ในไฟล์ DXSDK.ZIP จะเป็นชุดที่ทำการแก้บักเรียบร้อยและหลังทำการUnzip
เรียบร้อยแล้วให้ทำการติดตั้งและลง DPLAY50A.EXE ให้เรียบร้อย

หัวข้อโปรเจ็ก  การพัฒนาแอปพลิเคชันกราฟิกความเร็วสูงบนวินโดว์ด้วย DirectX
โดย 
	นายพิเชียร อนวัชพันธุ์
	นายสรวิชญ์ พลสิทธิ์
	ดร. บุญธีร์ เครือตราชู อาจารย์ที่ปรึกษา
	ปีการศึกษา 2540
