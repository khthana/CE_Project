Directory นี้จะเป็น code ของโปรแกรมที่ใช้ทดสอบและแสดงความสามารถของ DirectX
และสามารถรันได้ทันที เนื่องจาก Visual C++ได้แยกการ exe ไฟล์หลังการทำการ
compile ดังนั้นถ้าต้องการรันโปรแกรมให้ทำการเปลี่ยน directory ให้อยู่
directory เดียวกับโปรแกรมนั้นๆ และเวลาทำงานให้ทำการอ้างไฟล์ exe แบบ
relative กับ Directory ที่เราอยู่โดยทำดังนี้

cd man
release\man.exe

เพราะไฟล์พวก bmp mid และ wav จำเป็นต้องอยู่ที่เดียวกับโปรแกรมเวลาทำงาน
หรืออาจจะทำได้โดยทำการ copy ไฟล์ที่ใช้ในการรันโปรแกรม(ไฟล์ .exe )ไปไว้ที่
เดียวกับไฟล์ของรูปภาพและเสียงก็ได้
ก่อนที่จะทำการรันโปรแกรมเครื่องจำเป็นจะต้องติดตั้ง DirectX และ บางโปรแกรม
จำเป็นต้องมี Sound Card(โปรแรมส่วนมาก) และ Joypad หรือ Joystick
ถ้าไม่มีอุปกรณ์เหล่านี้จะทำให้โปรแกรมไม่สามารถทำงานได้

อธิบายโปรแกรม
SCROLL          เป็นโปรแกรมสำหรับแสดงการเลื่อนภาพขนาดเท่าหน้าจอแบบวน loop
MAN             เป็นโปรแกรมแสดงภาพตัวละครเป็นเคลื่อนไหว
BALL            เป็นโปรแกรมสำหรับทดสอบความเร็วในการการแสดงผล
EXAM1           เป็น code ของโปรแกรมตัวอย่างที่อยู่ในโปรเจ็ก
SCROLL2         เหมือนโปรแกรม SCROLL แต่สามารถใช้ joystick ได้
Hello_world     เป็นโปรแกรมแสดงภาพสามมิติ(เป็นตัวอย่างใน DirectX SDK)
Sunsystem       เป็นโปรแกรมที่ดัดแปลงจาก Hello_world
                โดยจะเป็นการจำลองระบบสริยะจักรวาล
