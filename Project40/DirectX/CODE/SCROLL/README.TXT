อธิบายโปรแกรม
เป็นโปรแกรมที่แสดงการเลื่อนหน้าจอซึ่งเป็นวิธีเดียวกับการเลื่อนของฉากในเกมส์ทั่วๆไป

ความสามารถใน DirectX
  -DirectDraw ในการแสดงผล  โดยใช้เทคนิก Double Buffer
  -DirectInput เพื่อใช้คีย์บอร์ดในการเลื่อนหน้าจอ

การควบคุม
  -ใช้คีย์ลูกศรในการเลื่อนหน้าจอ 
  -ESC เพื่อยกเลิกการทำงานของโปรแกรม

หมายเหตุ
	DirectInput ใช้อินเตอร์เฟสของเวอร์ชัน3 เพราะอินเตอร์เฟสของเวอร์ชัน 5 ไม่สามารถใช้กับตัวruntime ของเวอร์ชัน3ได้  ตอนที่เขียนโปรแกรมตัวนี้ตัว runtime มีเฉพาะในวินโดว์ 95 เท่านั้น  ในวินโดว์ NT ยังคงเป็นตัว runtime ของเวอร์ชัน 3อยู่  ทำเช่นนี้เพื่อให้โปรแกรมสามารถใช้ได้ทั้งบน 95และ NT
