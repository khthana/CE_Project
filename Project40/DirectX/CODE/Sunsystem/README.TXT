==================  sunsystem  ======================

อธิบายโปรแกรม
เป็นโปรแกรมสามมิติโดยใช้ Direct3D Retain Mode
เพื่อแสดงการเคลื่อนที่ในสภาพแวดล้อมสามมิติ
โปรแกรมคล้ายกับโปรแกรม hello_world แต่จะใช้เทคนิก hierachy
หลายออปเจ็กและเฟรมเข้าด้วยกัน
โดยโปรแกรมจะเป็นการจำลองการทำงานของระบบสุริยะจักรวาล

ความสามารถใน DirectX
  -ใช้ความสามารถของ Direct3D Retain Mode เป็นengin
  ในการคำนวนภาพสามมิติ  โดยจะใช้พื้นฐานทั่วไปในการอ้างอิงพิกกัดสามมิติ
  การแสดงแสง การหมุน การเคลื่อนที่ ในสภาพแวดล้อมสามมิติ
  การอ้างออปเจ็กและเฟรมหลายๆตัวเข้าด้วยกัน

การควบคุม
  -Up Arrow เดินไปข้างหน้า
  -Left Arrow หมุนตัวไปทางซ้าย
  -Right Arrow หมุนตัวไปทางขวา
  -Down Arrow ถอยหลัง
  -Numeric Pad 8 เลื่อนต้วขึ้น
  -Numeric Pad 4 เลื่อนต้วไปทางซ้าย
  -Numeric Pad 6 เลื่อนตัวไปทางขวา
  -Numeric Pad 2 เลื่อนตัวลง
  -PageUP หมุนตัวขึ้น
  -PageDown  หมุนตัวลง
  - "-" หมุนตัวทวนเข็ม
  - "+" หมุนตัวตามเข็ม
  -ESC เพื่อกลับมาที่ตำแหน่งเดิมในตอนเริ่มต้น
