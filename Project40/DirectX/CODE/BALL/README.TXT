==================  BALL  ======================

อธิบายโปรแกรม
เป็นโปรแกรมที่ใช้ในการทดสองความเร็วของการแสดงผลของกราฟฟิกสองมิติ  โดยทำการเพิ่มจำนวนของ Sprite หรือลูกบอลที่ปรากฎบนหน้าจอและทำการวัดจำนวนเฟรมที่แสดงผลเปรียบเทียบจำนวนsprite ที่แสดงผล

ความสามารถใน DirectX
  -DirectDraw ในการแสดงผล  โดยใช้เทคนิก Double Buffer
  -DirectInput ใช้คีย์บอร์ดในการควบคุมจำนวน Sprite ที่กำหนด
  -DirectSound ในการแสดงเสียงในลักษณะ wave

การควบคุม
  -กด insert เพื่อเพิ่มจำนวนลูกบอล
  -กด delete เพื่อลดจำนวนลูกบอล
  -ESC เพื่อยกเลิกการทำงานของโปรแกรม

หมายเหตุ
	ในโปรแกรมนี้ใช้ library CDX ซึ่งเป็น library ที่เรียกใช้ directx อีกทีหนึ่ง  ซึ่งเหมาะแก่การเขียนเกมส์  ที่นำมาใช้เพื่อลดระยะเวลาในการพัฒนาโปรแกรมนี้
	เนื่องจากDirectSound ไม่มีความสามารถในการเล่น MIDI ดังนั้นจะใช้ MCI ของวินโดว์ในการแสดงเสียงของ MIDI
