==================  MAN  ======================

อธิบายโปรแกรม
เป็นโปรแกรมที่ใช้การแสดงการทำ animation sprite ซึ่งเป็นเทคนิกเดียวกับการแสดงตัวละครที่เคลื่อนไหวได้ซึ่งใช้ในเกมส์ทั่วๆไป

ความสามารถใน DirectX
  -DirectDraw ในการแสดงผล  โดยใช้เทคนิก Double Buffer
  -DirectInput ใช้คีย์บอร์ดในการควบคุมการเคลื่อนไหวและการแสดงท่าทางของตัวละคร

การควบคุม
  -Ctrl ด้านซ้าย เตะ
  -Alt ด้ายซ้าย  ชก
  -Spacebar  กระโดด
  -Left Arrow ไปทางซ้าย
  -Right Arrow ไปทางขวา
  -Down Arrow นั่ง
  -ESC เพื่อยกเลิกการทำงานของโปรแกรม

หมายเหตุ
	ในโปรแกรมนี้ใช้ library CDX ซึ่งเป็น library ที่เรียกใช้ directx อีกทีหนึ่ง  ซึ่งเหมาะแก่การเขียนเกมส์  ที่นำมาใช้เพื่อลดระยะเวลาในการพัฒนาโปรแกรมนี้
