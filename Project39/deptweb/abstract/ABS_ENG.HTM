 ABSTRACT
"Department Intranet Project" is developed to study about posibility of applying computer 
system for information service in computer department. This system solve the past problem that comes 
from human mistakes to use computer collect documents and other information relating to acceptant with 
user inputing some infomation such as Id number , Title , Name of receivers and type of documents. 
Then , the system will automatically send letters and manipulate information storing in database for 
specific data and receiver can view their own and public information in the appropriate right in any time. 
This system will update database monthy to make appropriate disk space, not too big and user can adjust
some preferences such as list of receiver or types of data with themself.
