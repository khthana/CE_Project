บทคัดย่อ 
ปริญญานิพนธ์ฉบับนี้   เรียบเรียงขึ้นจากผลงานที่ได้พัฒนาระบบฐานข้อมูลเชิงสัมพันธ์แบบ
ไคลเอ็นท์เซิร์ฟเวอร์   เพื่อใช้ในการทำงานของระบบลงบัญชีคุมยอดการเบิกจ่ายเงินของแผนกการเงิน  ซึ่ง
ประกอบด้วยการกรอกข้อมูลเบื้องต้น  การบันทึกลงบัญชี  การจัดสรรเงิน  การสอบถามข้อมูล  และ
การพิมพ์รายงาน
การพัฒนาระบบฐานข้อมูลดังกล่าวใช้ระบบจัดการฐานข้อมูลเชิงสัมพันธ์ออราเคิล (  ORACLE )  
และใช้เดลไฟ ( DELPHI ) ในการเขียนโปรแกรมส่วนประมวลผลส่วนหน้า ( Front-End )  หรือส่วนที่ติดต่อกับ
ผู้ใช้ ( User Interface ) ให้เป็นแบบกราฟฟิค โดยมีไดรฟเวอร์โอดีบีซี  ( ODBC: Open  Database 
Connectivity )  ทำหน้าที่ในการติดต่อระหว่างส่วนประมวลส่วนหน้าและส่วนจัดการฐานข้อมูล   ในการพัฒนา
ระบบเริ่มจากการศึกษาระบบเดิมที่มีอยู่  ทำการวิเคราะห์  ออกแบบ   และติดตั้งระบบให้เหมาะสมกับการใช้งาน
