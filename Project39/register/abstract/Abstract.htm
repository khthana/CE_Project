Application of Database Management System for the Registrar’s Office in University

                              Nicom Panyapitipaitoon

                              Pichai Tawansangaram

                              Wiboon Prompanich Advisor

                              1995

Abstract

    This Application is developed for processing in Office of Central Register and Processing , time - table of class and class - room composing. This Application has useful for decrease mistakes that can occur and increase performance of processing. It has fast services that can support for number of users that can increase in the future. During the Implementation , we use three - tier Architecture that separate the Application in 3 parts Part 1 : Use Vistual Basic for Graphic User Interface ( GUI ). Part 2 : This part is OLE Automation Server and use Vistual Basic to create too. Part 3 : It is Data Part and we use Microsoft SQL Server to be a DBMS ( Database Management System ) and use ODBC ( Open Database Connectivity ) for connect to this part , that show you the Application can use Remote Automation for Control.
