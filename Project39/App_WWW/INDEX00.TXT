How to use FRIEND.ZIP

1.    First , you must have HTTP server for Windows on your machine.
2.    You must install Visual-Basic in this machine too. This example use Visual Basic 4.0
3.    In HTTP Server , there will be directory to store CGI application.
       This directory usually named "CGI-WIN"
4.    Create new directory in "CGI-WIN" for this CGI-Example
5.    Unzip file "FRIEND.ZIP" ,  you will get 
	CGI_MAIN.BAS
	CGI32.BAS
	STARTER.BAS
	FRIEND.VBP
	FRIEND.EXE
	LAEM.MDB
	LAEM.LDB
6.    .BAS files are Visual Basic module.  You MUST edit CGI_MAIN.BAS.
       You DO NOT do anything with CGI32.BAS and STARTER.BAS.
        LAEM.MDB and LAEM.LDB  is MS-Access database file.  This example will
        working with database in this file.

        You have to edit only 2 line in CGI_MAIN.BAS.
        In the declarations part , there are two constants  CGI_File and Database_File.
        You must edit this constant properly with your system.
