การพัฒนาความปลอดภัยของโปรแกรมเทลเนท

English Version

บทคัดย่อ
ปริญญานิพนธ์นี้นำเสนอการพัฒนาความปลอดภัยของโปรแกรมเทลเนทโดยโปรแกรมเทลเนทนี้ เป็นโปรแกรมอรรถประโยชน์นี้ใช้ในระบบเครือข่าย อินเตอร์เนต (internet) ซึ่งนำให้ผู้ใช้บริการ สามารถ login เข้ามาใช้บริการจากเครื่องอยู่ห่างไกลจาก Host นี้ที่ให้บริการระบบปฏิบัติการนี้ เลือกใช้ในโครงงานนี้ คือ ระบบปฏิบัติการ LINUX 2.0.0 สำหรับเครื่องให้บริการ (SERVER) และระบบปฏิบัติการ WINDOWS สำหรับเครื่องรับบริการ (CLIENT) การพัฒนาความปลอดภัย ของโปรแกรมเทลเนทนำได้โดยการเข้ารหัส และถอดรหัสข้อมูลนี้เป็นความลับของผู้ใช้บริการ เช่น login name และ password

โครงงานนี้นำการพัฒนาความปลอดภัยของโปรแกรมเทลเนทโดยเลือกใช้อัลกอริทึม เข้ารหัสแและ ถอดรหัส ด้วย RSA อัลกอริทึม เครื่องมือนี้ใช้ในการแปลภาษา ได้แก่ Microsoft Visual Basic 4.0 โดยนี้โครงงานนี้จะเป็นแนวทางในการพัฒนาระบบความปลอดภัยของโปรแกรมอรรถประโยชน์อื่นๆ ในระบบอินเตอร์เนทสำหรับผู้สนใจทั่วไป
