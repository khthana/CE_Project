DATA COMMUNICATION VIA

RADIO AMATEUR SYSTEM

        • Jirasak Pichaiyan
        • Veerawong Lohayuenyongsoong
        • Professor Somsak Mitatha Advisor
        • 1996
    Abstract
      Data communication via radio amateur system is the development 
    of software and hardware for communication via radio amateur system.
    The model of wireless data communication network consists of modems,
    microcomputer and FM transceiver.The first one is hardware concerning 
    about the 1200 bps modems.The second one is software concerning about 
    the data ttransferring program which can be used 5 function are multi 
    user mode,user to user mode,file mode,mail mode and morse mode.
