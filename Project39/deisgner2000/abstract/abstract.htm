Analysis and Design of an Information System using Oracle Designer/2000

                                  Walaiporn Parichatnont

                                  Sumonrat Tangjaijit

                                  Dr. Suphamitr Chittayasothorn (Advisor)

                                  1996

Abstract

This thesis is part of the project , Analysis and Design of an Information System, using Designer/2000 . The purpose of this project is to study how to approach application design and development using the Oracle Desinger / 2000 Repository and toolset. Designer / 2000 supports both structures,waterfall and spiral approches to development.The first stage of System Development with Designer/2000 is modelling business process with Process Modeller.This stage concerns with system definition study or strategic business study and know how business run. The next stage is to analyse business requirements in detail with system modeller such as Function Hierarchy Diagrammer . The third stage concern with application system and database design with System Designer including Data Diagrammer , Module Structure Diagrammer,Module Data Diagrammer and Module Logic Navigator .The Final stage is to build and implement these application and database with Generator. Designer/2000 CaseTool is appropriate to develop task because it supports both structures,waterfall and spiral approches to development.
