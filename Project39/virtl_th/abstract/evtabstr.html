1996

Abstract

Virtual Thailand Project is developed to give useful travel information of Thailand that Internet users can access this information by World Wide Web (WWW) service. In this project, it gives information about Thai lifestyles, fabulous festival, interested places, accommodations, today currency exchange rate, flight schedule and travellers’ tales.

Moreover, users can send his data to web server to get his desired information too, such as, users can use travellers’ tales function to tell the others about his Thailand trip, use currency exchange function to convert currency, use search function to find any places, use hotel reservation function to reserve any room, and use query function to get his desired place that meet his conditions. All the function mentioned above will give user useful information that can make up his mind to visit Thailand.
