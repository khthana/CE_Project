Abstract 
        This thesis is about a preseutation of queueing system on network which is developed and implemented to convenient   the users  in analysis the network   , which  helps on the calculating  to be more accurate  . The network queueing device can be provided into 3 main  parts , which are  the interface part  , the database part  and the network analysis part 
        All   the network queueing device is written in Delphi  , program is designed  by using Object - Domain and Visual Simnet for designing classes and the connections. The result of the analysis is program shows the flow of  data  that  gets in the system   .  Which  helps  the  system  designer has  a  better  plans  for  the  resource management.
