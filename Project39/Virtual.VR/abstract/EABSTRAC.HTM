Abstract

In present time, the picture of multimedia system usually presents in form of two-dimension.

Though we try to use many method to make the pictures look like three-dimension such as adjusting

bright and shadow but it doesn't work. Because of the monitors can display only two-dimension, that is

the problem. So we make this project. We apply the virtual reality theory to improve the quality of

picture by using three-dimension stereo display. User will see the real dimension of picture that showed

on the monitor. This project will reduce complex communication between user and picture. Three-

dimention is more easy to communicate than two-dimention. This project includes program for

controlling sequence of picture presentation. It use for controlling sequence of picture and attraction to

user.
