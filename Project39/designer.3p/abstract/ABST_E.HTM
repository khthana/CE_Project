Abstract

This thesis is a analysis and design using CASE tools. It can help us to do the work. The name of tools is Oracle Designer/2000. Oracle Designer/2000 is tool that help us to analysis, design and automatic generator. Since in the business system (simple, complex) needs the information system that have more efficient and joint with other tools such as DBMS of Oracle. Oracle Designer/2000 is a number of software toolsets for analyzing business requirement, and for designing client/server system that meet those business needs. Designer/2000 incorporates support for business process modeling, system analysis, software design and code generation.
