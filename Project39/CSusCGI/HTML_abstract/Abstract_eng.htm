Abstract  
        This project intends to propose an interesting alternative of database client/server system developing, by integrating it into World Wide Web (WWW) system and using Common Gateway Interface (CGI) program to link it with DataBase Management System (DBMS). An emphatic DataBase Management System is Oracle 7 Server.  
        Beside, this project is also related to an overviews of some of the tools that exist today to develope this database client/server system for every DataBase Management System and demonstrates the advantage and disadvantage of them.  
        At the end of this project is also related to the advantage and disadvantage of this database client/server system when compared with traditional database client/server system and demonstrates the applications that appropriate for database client/server using CGI system.
