Abstract

This project is the study of the Relational Database Management System or RDBMS (version 7.1.3.3.6) of ORACLE's RDBMS version 7.1.3.3.6 for improving the total effectiveness & performance of the system by using the finance DATABASE of Engineering Department as the DATABASE sample
This project demonstracts the system which is designed to be Client/Server system. The server's operating is Windows NT and The Client's operating system Windows95. Both parts connect and communication through SQL net, using TCP/IP protocol.
