Abstract

            This project propose a Wireless EPROM Wmulator thai is mostly used in EPROM's funtion simulation of microcontroller system . It;s used to store controlling program and control device functions.

            Additionally it's easy to download the program from microcomputer to this EPROM Emulator . So it's suitable for testing the frequently modified and developed program .
