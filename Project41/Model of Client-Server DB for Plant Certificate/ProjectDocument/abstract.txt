<p><b>บทคัดย่อ</b><p>
ปริญญานิพนธ์นี้เป็นส่วนหนึ่งของโครงงานพัฒนาเพื่อสร้างแบบจำลองของระบบฐานข้อมูล     สำหรับงานออกใบรับรองพืชที่จะทำการส่งออกนอกประเทศ
ของกองควบคุมพืชและวัสดุการเกษตร  สังกัดกรมวิชาการเกษตรโดยทำการทดลองวิเคราะห์และออกแบบระบบ   ( System Analysis and Design )   
ซึ่งเดิมที่เป็นระบบจะที่ใช้คนในการดำเนินงาน   เพื่อให้ระบบสามารถทำงานได้อย่างอัตโนมัติมากขึ้น  โดยอาศัยเทคโนโลยีสารสนเทศ ( Information Technology ) 
เข้ามาใช้งานให้มีประสิทธิภาพมากที่สุด 

โครงงานวิจัยนี้จะทำการออกแบบและสร้างฐานข้อมูลในรูปแบบฐานข้อมูลเชิงสัมพันธ์ ( Relational Database ) ไว้ที่ระบบให้บริการ( Server System ) 
ระบบจัดการฐานข้อมูลแบบสัมพันธ์( Relational Database Management System ) รวมทั้งทำการออกแบบและสร้างระบบลูกข่าย ( Client System )
สำหรับให้บริการ รับข้อมูลเข้าสู่ระบบ, แสดงผลข้อมูล, บริการพิมพ์ใบรับรอง และบริการพิมพ์รายงานสรุป <p>
<b>Abstract</b><p>
This thesis deals with the study of the Object - Oriented Concept to apply to the Relational database works. This
includes the analysis and making the comparison between Procedural Programming and Object - Oriented
Programming.

This study uses the Object - Oriented Language called Smalltalk to write the program that have a concept of the
Visual SQL program to search and give results by using the concept of SQL language.<p><hr>
