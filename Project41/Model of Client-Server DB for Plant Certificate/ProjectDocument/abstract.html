The Model of Client-Server Database for Plants Certificate

บทคัดย่อ

ปริญญานิพนธ์นี้เป็นส่วนหนึ่งของโครงงานพัฒนาเพื่อสร้างแบบจำลองของระบบฐานข้อมูล สำหรับงานออกใบรับรองพืชที่จะทำการส่งออกนอกประเทศ ของกองควบคุมพืชและวัสดุการเกษตร สังกัดกรมวิชาการเกษตรโดยทำการทดลองวิเคราะห์และออกแบบระบบ ( System Analysis and Design ) ซึ่งเดิมที่เป็นระบบจะที่ใช้คนในการดำเนินงาน เพื่อให้ระบบสามารถทำงานได้อย่างอัตโนมัติมากขึ้น โดยอาศัยเทคโนโลยีสารสนเทศ ( Information Technology ) เข้ามาใช้งานให้มีประสิทธิภาพมากที่สุด โครงงานวิจัยนี้จะทำการออกแบบและสร้างฐานข้อมูลในรูปแบบฐานข้อมูลเชิงสัมพันธ์ ( Relational Database ) ไว้ที่ระบบให้บริการ( Server System ) ระบบจัดการฐานข้อมูลแบบสัมพันธ์( Relational Database Management System ) รวมทั้งทำการออกแบบและสร้างระบบลูกข่าย ( Client System ) สำหรับให้บริการ รับข้อมูลเข้าสู่ระบบ, แสดงผลข้อมูล, บริการพิมพ์ใบรับรอง และบริการพิมพ์รายงานสรุป การสร้างฐานข้อมูลของระบบนั้นจะใช้ระบบจัดการฐานข้อมูล Microsoft SQL Server เป็นตัวจัดการ ในส่วนของการสร้างระบบลูกข่ายจะใช้เครื่องมือพัฒนาซอฟท์แวร์ ( Software Development Tool ) Microsoft Visual Basic เป็นตัวสร้าง

Abstract

This project is Client - Server database application delvelopment for plants certificate of the Agricultural Regulatory Division , Department of Agriculture , Thailand : The Model of Client-Server Database for Plants Certificate. It's analysis and design the system is operated by the people in the past , to make the system worked more automatically and most efficiently by using Information Technology. And it's designed , constructed to client system served for receiving , showing information to the system , printing phytosanitary certificate and summarized report. For delveloping , using Microsoft SQL Server act DBMS and Microsoft Visual Basic ' software delvelopment tool to construct the client system.
