ABSTRACT
 
 

                        This project is implemented by Multimedia ToolBook program , Paradox program and  3D Studio program for presenting picture of activity of KING MONGKUT’S INSTITUTE OF TECHNOLOGY LADKRABANG. Multimedia ToolBook program can be used for tasks that present vidio , audio and animation activities as well. Each content of chapter consist of creating objects  ,  linking a database , technique and implementing , and how to use the program.  A program source code is included in the Appendix .  The result ‘s  project  can  be used to store and search various kinds of picture database.
