Abstract

This project deals with the study of Network Management System and using Simple Network Management Protocol (SNMP) for developping and implementing of an agent on network’s devices. These devices come from mutivendors. Informations embeded in the agent could be retrived from any Network Management Station (NMS). From these informations, the administrator shall be properly notified when there are status changed in any network equipment also, managing can be done remotely.
