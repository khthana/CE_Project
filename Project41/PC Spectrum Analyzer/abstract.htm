บทคัดย่อ

ปริญญานิพนธ์นี้ได้นำเสนอการนำเครื่องไมโครคอมพิวเตอร์มาวิเคราะห์แถบความถี่ของสัญญาณ โดยใช้อุปกรณ์ต่อร่วมในการทำงาน คือ ชุดพัฒนา DSP Start Kit ซึ่งเป็นชุดไมโครคอนโทรลเลอร์ที่มีโปรแซสเซอร์ TMS320C50 เป็นตัวประมวลผลกลาง ทำหน้าที่รับสัญญาณอนาล็อคที่ต้องการนำมาวิเคราะห์แปลงเป็นสัญญาณดิจิตอลแล้วนำมาวิเคราะห์หาแถบความถี่โดยโปรแกรม FFT จากนั้นส่งผลคำตอบที่ได้ส่งไปแสดงผลยังเครื่องไมโครคอมพิวเตอร์โดยการแสดงผลจะแสดงเป็นภาพกราฟิกเพื่อง่ายต่อการอ่านและแสดงผลเป็นตัวเลขเพื่อการอ่านที่แม่นยำถูกต้องตามค่าจริงที่วัดได้

Mr. Phaitoon Srinil

Mr. Ackachai Chaiyakool

______________ Advisor

ABSTRACT

This project proposes to apply the microcomputer to the Spectrum Analyzer by used peripheral is DSK board which is the microcontroller TMS320C50. Input the analog signal to DSK board and convert to digital signal, then analyze the spectrum by FFT Program. Transfer result to microcomputer and then display the graphics for user-friendly and exactly numeric.
