เมื่อลง INFORMIX เรียบร้อยแล้ว
1. จะต้องทำการสร้างฐานข้อมูลขึ้นโดยเข้าไปใน Directory Database
     การสร้าง ROW TYPE ของ DATA FLOW และ ER DIAGRAM 
     โดยเรียกไฟล์ DataFlowRowType และ ERRowType  ขึ้นมาแล้ว RUN ที่ SQL EDITOR
     จากนั้นจึงทำการสร้างตารางโดยเรียกไฟล์ DataFlowTable และ ERTable ขึ้นมาแล้ว RUN
2. สร้างรูทีนที่ใช้ทั้งหมดโดยเรียกไฟล์ต่างๆที่อยู่ใน Directory Routine ขึ้นมาแล้วทำการ RUN 
    ที่ SQL EDITOR
3. การเรียกไฟล์แอพพลิเคชัน สามารถเรียกไฟล์ ddf1.exe ใน Directory Application 
    ขึ้นมาใช้งานได้ทันที
