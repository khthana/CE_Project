บทคัดย่อ
 	การเขียนโปรแกรมเชิงวัตถุถือได้ว่าเป็นเทคนิคที่ช่วยให้การพัฒนาโปรแกรมเป็นไปได้ง่ายขึ้น เนื่องจากความสามารถในการนำกลับมาใช้ใหม่ได้ หลักการเขียนโปรแกรมเชิงวัตถุจะประกอบด้วยความสามารถในการซ่อนข้อมูล ความสามารถในการสืบทอด และความสามารถในการเป็นได้หลายรูปแบบ ในภาษาจาวานั้นก็ใช้เทคนิคการเขียนโปรแกรมเชิงวัตถุเช่นกัน ภาษาจาวาสร้างขึ้นมาจากคุณสมบัติที่มีประโยชน์ของภาษาซีพลัสพลัสและมีโครงสร้างคล้ายกับภาษาซีพลัสพลัส แต่จะกำจัดความซับซ้อน ความเสี่ยง และส่วนประกอบที่ไม่เป็นประโยชน์ออกไป ทำให้ได้ภาษาที่มีความปลอดภัย เข้าใจได้ง่าย และง่ายต่อการใช้ ภาษาจาวาสามารถทำงานโดยไม่ขึ้นกับแพลตฟอร์ม สนับสนุนมัลติเธรด ใช้สร้างเว็บได้ และยังสามารถทำงานร่วมกับแอ็กทีฟเอ็กซ์ได้
Abstract
	Object-Oriented Programming is a new technology that makes the programming easier because of its reusibility. Object-Oriented Programming has capability encapsulation, inheritance, and polymorphism. Java is Object-Oriented Programming. Java builds on the familiar and useful features of C++ while removing its complex, dangerous, and superfluous elements. The result is a language that is safer, simpler, and easier to use. Java is independent platform, multithread, and support programming web. In addition, it can integrate with ActiveX.
