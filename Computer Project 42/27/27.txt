Abstract
	There are five phases of system development : requirement analysis, analysis, design, programing and                              testing.  In system development with UML, It sees various views of system. Which are different.  There are use-case view, logical view, component view and deployment view. Each view can be explained by diagrams.  There are nine types of diagrams : use-case diagram, class diagram, object diagram, state diagram, sequence diagram, collaboration diagram, activities diagram, component diagram and deployment diagram.  Also have the example of usage.
