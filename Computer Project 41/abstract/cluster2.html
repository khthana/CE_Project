Cluster Computing Using PVM (Parallel Virtual Machine)

บทคัดย่อ

การประมวลผลแบบกลุ่ม (Cluster computing) นี้เป็นอีกแนวทางหนึ่งในการพัฒนาการประมวลผลของ คอมพิวเตอร์ให้มีความเร็ว และประสิทธิภาพที่สูงขึ้น ซึ่งในการศึกษาครั้งนี้จะมุ่งเน้นไปที่โปรแกรมที่ทำให้เกิดการ ประมวลผลแบบรวม โดยโปรแกรมที่จะนำมาศึกษานี้คือ PVM (Parallel Virtual Machine) โดยมีคุณสมบัติทำให้ คอมพิวเตอร์ที่มีสถาปัตยกรรมที่ต่างกันสามารถทำงานร่วมกันได้โดยใช้การเชื่อมต่อทางระบบเครือข่ายแบบต่างๆ เช่น Ethernet, FDDI และอื่นๆ เสมือนเป็นคอมพิวเตอร์เครื่องเดียว ซึ่งแน่นอนว่าประสิทธิภาพในการทำงานย่อมสูง กว่าการทำงานของคอมพิวเตอร์เพียงเครื่องเดียวอย่างแน่นอน ในการปรับแต่งองค์ประกอบต่างๆของการประมวลผลแบบกลุ่ม โดยใช้โปรแกรม PVM นี้สามารถทำได้ ง่ายและยังสามารถสร้างการประมวลผลแบบอนุกรม ขนาน และแบบ Vector computers ได้ทำให้โปรแกรม PVM นี้ได้รับความนิยมอย่างรวดเร็วและแพร่หลาย โดยยกตัวอย่างจาก Sun Cluster ซึ่งมีการทำงานแบบ Cluster และม ีการสร้างทั้งทางด้าน Hardware และ Software ขึ้นมาใช้เองทั้งหมดและทางด้าน Software นั้นยังสนับสนุนเครื่อง คอมพิวเตอร์ที่มีสถาปัตยกรรมอื่นด้วย ซึ่งการทำการประมวลผลแบบรวมนี้มีประโยชน์ที่น่าสนใจและน่าศึกษานำ ไปใช้ได้ดี

Abstract

Cluster computing is another method to increase speed and efficiency of computation. The point of this project is study in PVM program that enables a collection of heterogeneous computer to be as a coherent and flexible concurrent computational resource that may be interconnected by variety of networks such as Ethernet, FDDI etc. Actually, efficiency and computational power have been increase more than a single computer in set of cluster. Additional, In configuration of PVM program simply adjust the collection of serial, parallel, and vector computer for this reasons it cause PVM is popularity is quickly and widespread. For example the Sun cluster that work in cluster method which have developed his own hardware and software. And his model can be used for other architecture that is interesting for study.
