บทคัดย่อ

รายงานนี้เป็นรายงานที่ค้นคว้าเกี่ยวกับเรื่องเทคโนโลยีทางซอฟท์แวร์ของบริษัทไมโครซอฟท์ ที่มีชื่อว่า "Active X " ซึ่งทางบริษัทไมโครซอฟท์ต้องการให้เป็นมาตราฐานทางซอฟท์แวร์ในปัจจุบัน ซึ่ง Active X จะเป็นเรื่องที่เกี่ยวกับการใช้โปรแกรมร่วมกัน เช่นการที่ผู้ใช้ต้องการที่จะนำเอาแฟ้มเอกสารที่สร้างจากโปรแกรม VISIO มาใช้ร่วมงานกับโปรแกรม Microsoft Word และสามารถที่จะทำการแก้ไขข้อมูลได้จากโปรแกรม Microsoft Word ได้เลยทันทีและมีการใช้นำเอา Active X มาใช้งานบนอินเทอร์เน็ตด้วย

Abstract

This report represents software technology of Microsoft corporation , that called " Active X " : Microsoft corporation desire this software technology to be standard recently. Active X is the software technology about using several programs simultaneously ; such as while using Microsoft Word , one can load visio file to work together and can improve visio file on Microsoft Word. In addition Active X is applied to use for Internet.
