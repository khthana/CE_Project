บทคัดย่อ

เนื่องจากว่าในยุคปัจจุบันนี้เป็นยุคของเทคโนโลยีสารสนเทศ การสื่อสารรับส่งข้อมูลต่างๆ ต้องการความ รวดเร็วมากขึ้น ทำให้การบีบอัดข้อมูลเริ่มเข้ามามีบทบาทสำคัญและเป็นที่แพร่หลายมากขึ้นกว่าเดิม เช่น การดาวน์ โหลดไฟล์ ทั้งภาพ เสียง อักขระ หรือไฟล์โปรแกรมต่างๆจาก อินเตอร์เน็ต หรือการสำรองข้อมูล ซึ่งวิธีการบีบอัด ข้อมูลนี้ช่วยให้การรับส่งข้อมูลต่างๆ ไม่ว่าจะเป็นภาพเสียงหรือตัวอักษร ทำได้เร็วขึ้นอีกทั้งยังช่วยประหยัดพื้นที่อีกด้วย โครงการนี้จะเป็นการศึกษาเกี่ยวกับเรื่องของการบีบอัดข้อมูลในรูปแบบต่าง ๆ เช่น การเข้ารหัสแบบ Shannon-Fano, การเข้ารหัสแบบ Huffman, การเข้ารหัสแบบ Adaptive Huffman, การเข้ารหัสแบบ Arithmetic, การบีบอัดแบบ LZ78 และ ศึกษาประสิทธิภาพการทำงานของโปรแกรมการบีบอัดข้อมูล 4 โปรแกรม รวมทั้งสรุป เปรียบเทียบการทำงานของทั้ง 4 โปรแกรม

Abstract

Today, world is changed to information technology times. All of communications need much more speed. So data compression have an importance part and to be used widely. Such as downloading file from Internet or backup data which data compression method will help data can be transmitted more speed and save space to storage too. This project is emphasized for educate several of data compression methods like Shannon-Fano Coding, Huffman Coding, Adaptive Huffman Coding, Arithmetic Coding, LZ78 Compression and finally compare performance of four data compression program and conclusion.
