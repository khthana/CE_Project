บทคัดย่อ

Common Gateway Interface(CGI) มีการพัฒนาหนทางนำไปสู่ การแสดงผลแบบ dynamic ของข่าวสารบน World Wide Web เป็นรูปแบบแรก CGI ที่ทำงานบนระบบคอมพิวเตอร์นั้นจะทำการสร้าง(generate) Web pages โดยทันทีหลังจากที่ได้รับการร้องขอจากผู้ใช้ CGI สามารถทำความเข้าใจวิธีการต่างๆ ในการสร้าง pages ได้ง่าย และสามารถเรียกใช้ฟังก์ชั่นต่างๆ ของเครื่องให้บริการ(server ) ได้ เช่นฟังก์ชั่นที่ใช้ในการเรียกดูวันที่หรือเวลาของเครื่องให้บริการ(server) ในการเขียนโปรแกรมในรูปแบบ CGI นั้นเราสามารถใช้ภาษาคอมพิวเตอร์ได้หลายภาษามาทำการเขียนได้ เช่น ภาษาซี ภาษา Perl หรือแม้แต่ภาษายอดฮิตอย่าง Visual Basic ก็สามารถนำมาใช้ได้ แต่ภาษาที่หลายๆ คนชอบใช้คือ ภาษา Perl การประยุกต์ใช้ที่โดดเด่นของ CGI คือการสร้างเว็บเพจที่สามารถรับข้อมูลจากผู้ใช้ ที่เรียกกันว่าการสร้างฟอร์ม หน้าตาของฟอร์มนั้นเราสามารถใช้ภาษา HTML ในการสร้างได้ แต่หลังจากที่ผู้ใช้ทำการกรอกข้อมูลเข้าไปในฟอร์แล้วจะเป็นหน้าที่ของ CGI ที่จะนำข้อมูลนี้ไปทำงานอื่นๆ และ CGI ยังสามารถนำไปประยุกต์ใช้งานอื่นๆ ได้อีก

Abstract

The Common Gateway Interface(CGI) emerged the first way to present dynamically generate information on the World Wide Web. CGI well-understood method for creating such pages. CGI can call functions on computer server ,for example function call to display date and time on the computer server. Programming in CGI can write by several language , such as C, PERL, or Visual Basic. This language can writing code to CGI program but popularly language is Perl. Applied in CGI at prominent is FORM ,feature of FORM can create by HTML but after user put all information on FORM next step is role of CGI .To be execute information in work others. And CGI can apply to other application.
