บทคัดย่อ

โครงงานนี้ทำการศึกษาเกี่ยวกับโปรแกรมการสื่อสารบนอินเตอร์เน็ตที่ได้รับความนิยม ขั้นตอนแรกทำการศึกษาโปรแกรมในด้านลักษณะการใช้งาน หลักการทำงาน ความสามารถ และ คุณลักษณะที่สำคัญในระดับผู้ใช้งาน ( user ) จากนั้นขั้นตอนที่สองเป็นขั้นตอนการทดลอง ทำการแบ่งโปรแกรมออกเป็นสองกลุ่ม คือ กลุ่มแรกเป็นโปรแกรมที่สื่อสารด้วยข้อความ กลุ่มที่สองเป็นโปรแกรมที่สื่อสารด้วยวิดีโอและออดิโอ แล้วทำการเปรียบเทียบความสามารถและคุณลักษณะในด้านต่างๆของแต่ละกลุ่ม ขั้นตอนสุดท้ายทำการสรุปผลที่ได้จากการศึกษา

Abstract

This Project is study about the communication softwares on Internet that also popular . First step is studying about how to use them, how they work, theirs ability that provide and theirs features in the level of user. Second step is the experiment. The experiment devides them in two groups. First group is text-based softwares. Second group is video and audio softwares. Comparing abilitys and features in each view in each group. Last step is to summarize the study.
