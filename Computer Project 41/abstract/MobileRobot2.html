บทคัดย่อ

รายงานนี้จะมีเนื้อหาครบคลุมเกี่ยวกับเรื่องแขนกล ซึ่งจะกล่าวถึงชนิดของแขนกล และส่วนที่จำเป็นของแขนกลคือมือกล ซึ่งแบ่งออกเป็นมือที่ใช้ในการจับ กับมือที่ใช้เป็นอุปกรณ์ เช่น เครื่องพ่นสี และ เครื่องเชื่อม ส่วนถัดไปจะกล่าวถึงคุณสมบัติของเซ็นเซอร์ชนิดต่างๆ เช่น เซ็นเซอร์สัมผัส , เซ็นเซอร์อินฟราเรด และเซ็นเซอร์แอลตร้าโซนิก ส่วนสุดท้ายจะกล่าวถึงแขนกลที่มือมีลักษณะคล้ายกับมือของมนุษย์

Abstract

This project is concern with robot arm. One common structural classification of robot arms. And essential of a robot arm is end - effector which can be either a gripper or some sort of tool such as a paintsprayer or welding gun. Next part are mention with sensor such as touch sensor, infrared sensor and ultrasonic sensor, Finally mention with robot hand.
