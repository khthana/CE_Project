บทคัดย่อ

โครงงานนี้เป็นการศึกษาโครงงานของรุ่นพี่ที่เกี่ยวกับระบบรักษาความปลอดภัย ซึ่งเป็นการนำ Microcomputer, Microcontroller และการรับ-ส่งข้อมูลเครือข่าย Computer มาประยุกต์เป็นระบบรักษาความปลอดภัยในหมู่บ้าน โดยแบ่งเป็น 2 ส่วน ส่วนแรกเป็นระบบเตือนภัยของบ้านแต่ละหลัง ใช้ Microcontroller MCS-51 เป็นส่วนประมวลผลกลาง ติดต่อกับอุปกรณ์ตรวจจับ (Sensor) ได้สูงสุด 32 จุด ควบคุมอุปกรณ์ไฟฟ้าได้ 4 จุด ส่งสัญญาณแจ้งเตือนทาง Siren และต่อโทรศัพท์แจ้งเตือนเจ้าของบ้านหรือสถานีตำรวจโดยอัตโนมัติ ส่วนที่ 2 เป็นระบบเครือข่ายแม่กับลูก 2 ระดับ ใช้ Microcomputer ติดตั้งที่ศูนย์รักษาความปลอดภัยของหมู่บ้าน ทำหน้าที่แสดงผลการทำงานและส่งสัญญาณข้อมูลควบคุมระบบทั้งหมดแบบอนุกรมอ้างอิงมาตรฐาน RS-485 นอกจากนี้ยังได้ทำการศึกษาระบบรักษาความปลอดภัยของต่างประเทศ ซึ่งในที่นี้ได้ทำการศึกษาระบบ 3500 ของบริษัท Digitize ซึ่งเป็นระบบเตือนภัยที่จะคอยรับ Input จากอุปกรณ์เตือนภัยในจุดต่าง ๆ ซึ่งในระบบนี้จะมี Serial Port ติดมาให้ 5 Port (สามารถเพิ่มได้ 33 Port) และยังสามารถทำการเชื่อมต่อระบบนี้ให้ใช้ร่วมกันได้ถึง 99 ระบบ และในตอนท้ายยังได้ทำการเปรียบเทียบระบบรักษาความปลอดภัยทั้ง 2 ระบบที่ได้ทำการศึกษามา ว่ามีความสามารถในการทำงานแตกต่างกันอย่างไร

Abstract

This project is studying the previous project about security system that using microcomputer, microcontroller and data transmission via computer network for application of village security system. This system consists of two part, the first is a alarm system for each house using MCS-51 microcontroller to be CPU for receiving 32 sensor devices and control 4 electric devices and sending alarm signal with siren sound and automatic call to house owner or police station. The second part is Master / Slave network using microcomputer in village security system for indicating an alarm display and sending all of control data of system referenced by serial communication RS-485 standard. Otherwise, this project, study the foreign security system especially system 3500 by Digitize, Inc. This system is alarm system that accepts input from virtually any remote alarm device with have 5 serial ports standard (expandable to 33) and can be interconnected with 99 system. Finally, in the last section of this project is comparing 2 security system for the difference of functional ability in 2 system.
