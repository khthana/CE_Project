บทคัดย่อ

เทคโนโลยีมัลติมีเดียได้ถูกนำไปใช้งานอย่างกว้างขวางในงานหลาย ๆ ด้าน ทำให้ในปัจจุบันได้มีการพัฒนา ระบบมัลติมีเดียไปอย่างรวดเร็ว จนกระทั่งได้มีการพัฒนาระบบมัลติมีเดียเข้ากับระบบเครือข่ายสื่อสาร ในโครง งานนี้จะได้กล่าวถึง เครือข่ายมัลติมีเดีย ซึ่งส่วนประกอบที่สำคัญที่สุดอย่างหนึ่งของเครือข่ายมัลติมีเดียก็คือ เครื่อง คอมพิวเตอร์ที่สามารถสนับสนุนระบบมัลติมีเดีย รวมทั้งซอฟต์แวร์ ซึ่งสามารถแบ่งออกได้เป็น 3 ประเภทคือ MCI , TAPI และ API ในการนำเครือข่ายมัลติมีเดียไปประยุกต์ใช้งาน เช่น การประชุมทางไกลผ่านทางจอภาพ ซึ่งจำ เป็นที่จะต้องมีมาตรฐานเพื่อใช้ในการกำหนดฟังก์ชั่นหน้าที่ต่าง ๆ เช่น H.320 , H323 , H324 เป็นต้น ในการส่งข้อ มูลผ่านเครือข่ายมัลติมีเดียจำป็นต้องใช้เทคนิคการบีบอัดข้อมูล เพื่อทำให้ขนาดข้อมูลที่จะทำการส่งมีขนาดเล็กลง โดยใช้มาตรฐาน H.261 ในการเข้ารหัส , การถอดรหัส และการบีบอัดข้อมูล ในการประยุกต์ใช้งานเครือข่าย มัลติมีเดียเราจำเป็นจะต้องพิจารณาถึงเครือข่ายที่ใช้ในการสื่อสาร ซึ่งมีมากมายหลายชนิดเช่น ISDN นอกจากนั้น แล้วยังมีรูปแบบการส่งข้อมูลดิจิตอลวีดีโอแบบอื่น ๆ อีก เช่น ดาวเทียม คลื่นไมโครเวฟ เครือข่ายโทรศัพท์ สาย เคเบิล และการกระจายสัญญาณบนพื้นดิน การประชุมทางไกลผ่านจอภาพสามารถนำไปประยุกต์ใช้ในงานด้าน ต่าง ๆ และซอฟต์แวร์ที่ใช้ก็มีหลากหลายชนิดเช่น CINE Video/Direct, MeetingPoint

Abstract

Multimedia technology was used in many kind of work, so they have a lot of development in a short time. Until multimedia technology and telecommunication technology were combined together. In this project presents the specification of multimedia network. The essential component of multimedia network is multimedia PC includes the software environment that can divide in three parts (MCI, API and TAPI). In multimedia network application, each video conferencing, have necessarily standard to define functions for some tasks (H.320, H.323, H.324). Compression technology must be used to compress the transmission data to resize transmission data. Encoding, decoding and compressing data were performed by H.261 standard. In multimedia network, we must consider about media, that were used for transmit data such as ISDN. Not only that, they have the Digital Video Broadcasting for transmit digital data through many kind of media such as satellite, microwave, telephone system, cable, terrestrial broadcasting. One application of multimedia network is video conferencing that can apply to a lot of works. There is a lot of software for video conferencing such as CINE Video/Direct, MeetingPoint.
