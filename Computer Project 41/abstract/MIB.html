บทคัดย่อ

Management Information Base (MIB) เป็นฐานข้อมูลที่ใช้เก็บข้อมูลในการบริหารระบบ ช่วยให้ผู้บริหาร ระบบเครือข่ายทราบถึงความเป็นไปของระบบรวมถึงสามารถนำข้อมูลมาช่วยในการวิเคราะห์ระบบเครือข่าย ในเนื้อหากล่าวถึง โครงสร้างของ MIB ,MIB มาตรฐาน ซึ่งแบ่งเป็น 11 กลุ่ม ,Enterprise MIBs ซึ่งเป็นข้อมูล ที่ผู้ผลิตกำหนดขึ้นมาเอง ,MIB Module ,จุดประสงค์ของการบริหารระบบรวมทั้งในการใช้ ASN.1 ในการกำหนดรูป แบบของ MIB

Abstract

Management Information Base (MIB) is the database for storing data in management system. In network system, it not only can show the working condition but also analyst the system. All the following are included in this report : 1. Structure of MIB 2. Standard of MIB, which is divided into 11 groups 3. Enterprise MIBs ,which is determined by the producer 4. MIB Module 5. The purposes of management system 6. ASN.1 ,the instructions to determine MIB features
