บทคัดย่อ

ไมโครคอนโทรลเลอร์ 68HC11 เป็นชิปไมโครคอนโทรลเลอร์ขนาด 8 บิตของโมโตโรลา ที่ได้ถูกออกแบบมาโดยรวมอุปกรณ์ที่จำเป็นในการใช้งานของระบบไมโครคอนโทรลเลอร์ไว้ภายในตัวของมันเอง ไม่ว่าจะเป็น พอร์ตอิน-พอร์ตเอาต์พุต วงจรตั้งเวลา ตัวนับระบบการอินเตอร์รัปต์ หน่วยความจำรอม แรม และอีอีพรอม วงจรแปลงสัญญาณแอนาล็อกเป็นดิจิตอล ระบบป้องกันความผิดพลาดที่จะเกิดขึ้นจากโปรแกรม และส่วนติดต่อสื่อสารข้อมูลอ่านทางพอร์ตอนุกรม ไมโครคอนโทรลเลอร์ MC68332 เป็นไมโครคอนโทรลเลอร์ของบริษัท Motorola ที่วางตลาดเพื่อจุดประสงค์ให้เป็นไมโครคอนโทรลเลอร์ขนาด 32 bit โดยมี internal data bus ขนาด 32 bit ,register ต่างๆมีขนาด 32 bit แต่อย่างไรก็ตามไมโครคอนโทรลเลอร์ตัวนี้มี External data bus ขนาด 16 bit , address ขนาด 24 bit จึงจัดไมโครคอนโทรลเลอร์ตัวนี้ไว้เป็นไมโครคอนโทรลเลอร์ 16 bit ซึ่งส่วนประกอบที่สำคัญของไมโครคอนโทรลเลอร์ตัวนี้มี 5 ส่วนคือ 1. Central processing unit (CPU32) 2. System integration module(SIM) 3. Time processing unit (TPU) 4. Queued serial module (QSM) 5. Static RAM module with TPU Emulation Capability (TPURAM)

Abstract

Microcontroller 68HC11, a 8-bit microcontroller, combines high-performance data manipulation capabilites with powerful peripheral subsystems such as input-output port, programmable timer, real-time interrupt , programmable memory, ADC, and Serial communication Interface and Serial Peripheral Interface. Microcontroller MC68332 , motorola products, is produced to be a 32-bit microcontroller. It has 32-bit internal databus, 32-bit register however it has 16-bit external databus and 24-bit address bus. So this is a 16-bit microcontroller. There are five most important combination , Central processing unit (CPU32), System integration module(SIM), Time processing unit(TPU), Queued serial mudule(QSM), and Static RAM module with TPU Emulation Capability (TPURAM).
