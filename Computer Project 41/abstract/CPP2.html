บทคัดย่อ

ในส่วนของ dynamic object creation นั้น จะเป็นการสร้างอ็อบเจคที่สามารถลบได้ในช่วงเวลา run-time เพื่อการรองรับข้อมูลที่เราไม่สามารถทราบข้อมูลที่แน่นอนของมันได้โดยการใช้ฟังก์ชันของ C++ คือ ฟังก์ชัน new และฟังก์ชัน delete เพื่อใช้ในการจองเนื้อที่สำหรับอ็อบเจค และยังได้กล่าวถึงการโอเวอร์โลดฟังก์ชันเพื่อการใช้งานที่กว้างขวางอีกด้วย การทำ virtual function จะเป็นการสร้าง function ที่รองรับการทำงานของ class ในการสืบทอด (Inheritance) ซึ่งการทำ virtual ถือเป็นส่วนประกอบสำคัญในการเขียนโปรแกรมภาษาที่มี Object-oriented programming ซึ่งเป็นคุณสมบัติของ C++ ที่เพิ่มเติมขึ้นมาจาก C หรือ Turbo C โดยอาศัยความสามารถของ compiler ที่มากขึ้นในการใช้ในการชี้ถึงตำแหน่งของ function หรืออื่นๆ และในการสร้าง constructor หรือ destructor ของ function ต่างๆ

Abstract

Dynamic object creation is the object that created and deleted in run-time. It useful when we could not know the number of the object created exactly. There are two functions in C++ new and delete for allocate and release memory for the object and there is overloading function for the widely use Function in inheritance is multiplex, So the virtual function is necessary for the difference code in programming. The virtual function is one of the essential characteristics of Object-oriented Programming that is the additional function of C++ form C or Turbo C which compiler is have more efficient to compile the source code to use pointer (EX. vpointer , VTABLE) include constructor and destructor.
