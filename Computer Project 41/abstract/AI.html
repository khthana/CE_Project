บทคัดย่อ

ปัญหาที่พบในการวิจัยและทดลองการรู้จำเสียงอย่างหนื่งก็คือข้อมูลเสียงที่ใหญ่มากทำให้เปลืองเนื้อที่หน่วยความจำ และเวลาที่ใช้ในการประมวลผล รายงานฉบับนื้ได้ศึกษาวิธีการลดขนาดข้อมูลเสียงโดยการแปลงข้อมูลแบบคาร์ฮูเน็นเลิฟ วิธีการนี้สามารถที่จะลดขนาดของข้อมูลขนาดใหญ่ที่เกี่ยวเนื่องกัน ให้เหลือเพียงข้อมูลขนาดเล็กที่เป็นอิสระต่อกัน ซึ่งจากการศึกษาโดยใช้วิธีนี้ มีความเป็นไปได้ที่จะนำมาประยุกต์กับการลดขนาดข้อมูลเสียงในแกนเวลาผลที่ได้นอกจากลดขนาดข้อมูลแล้ว ข้อมูลที่ได้ยังมีความเหมาะสมที่จะนำไปใช้ในขั้นตอนการรู้จำเสียงพูดภาษาไทย

Abstract

For a long time , there are the development in speech recognition system , the one thing is an obstacle of this project is the large amount of input speech data . Because it consume a lot of memory spaces and spend processing times of CPU . In this paper is the study of Reducing Speech Data by Karhunen-Loeve. The large data that correlated will be reduced to smaller data .It also can reduce the large amount of speech data in time domain. After transform can get new domain data then it can use in speech recognition system.
