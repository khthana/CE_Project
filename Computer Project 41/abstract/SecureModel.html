บทคัดย่อ

บทความฉบับนี้ กล่าวถึงสถาปัตยกรรมของรูปแบบการจัดตั้งระบบรักษาความปลอดภัย ซึ่งประกอบไปด้วยเรื่องราวเกี่ยวกับ ปัญหาด้านความปลอดภัยในระบบ แนวทางการออกแบบระบบรักษาความปลอดภัย และขั้นตอนในการสร้างระบบรักษาความปลอดภัย ซึ่งจะเริ่มตั้งแต่การประเมินและการวางแผนทางด้านระบบรักษาความปลอดภัย การออกแบบระบบรักษาความปลอดภัย การจัดตั้งระบบรักษาความปลอดภัย ไปจนถึงการจัดการกับระบบรักษาความปลอดภัยเมื่อนำไปใช้งานจริง

Abstract

The text in this article is written about the Security Model Architecture which comprises the details as follows : the problems concerning the system security, basis used in designing the security system, stages in implementing the security system begin with Security Assessment & Planning Methods, Security Architecture & Design Methods, Security Implementation Methods, as well as Security Management Methods when the system is being implemented.
