บทคัดย่อ

รายงานนี้กล่าวถึงระบบเครือข่ายของคอมพิวเตอร์ภายในสถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบังว่ามีการเชื่อมต่อและมีการให้บริการแก่อาจารย์ เจ้าหน้าที่ นักศึกษา ตลอดจนผู้สนใจ โดยได้มีการกระจายการบริการไปยังอาคารต่างๆ ทั่วทั้งสถาบัน ระบบเครือข่ายของสถาบันฯจะประกอบด้วย network 2 ส่วน ที่เชื่อมต่อกัน คือ KMIT'L Campus Network ,Open System Network บริการที่มีอยู่ในเครือข่ายมีอยู่มากมาย เช่น บริการบนเครื่องเจ้าคุณ ซึ่งจะเกี่ยวกับทางด้าน Internet ทั้งหมด, บริการบนเครื่องแคแสด ก็จะเกี่ยวกับ Application Software ต่าง ๆ ที่จะสามารถเข้าไปใช้งานได้, บริการค้นหาข้อมูลสารสนเทศในห้องสมุด ฯลฯ นอกจากนี้ยังมีระบบเรียนการสอนทางไกลแบบ 2 ทาง ระหว่าง สถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบัง กับ วิทยาเขตชุมพร และ โครงงานวิจัยร่วมระหว่าง ไทย จีน และญี่ปุ่น เพื่อพัฒนาทรัพยากรมนุษย์

Abstract

This report represent KMIT'L campus network how to connect to all buildings for serving teachers, students, authorities and interest people. KMIT'L consists of two parts :KMIT'L Campus Network, Open System Network KMIT'L Campus Network has several services such as Chaochun server service which serves for internet, Khaesad server service which serves for application softwares which can access to use, service for search about information technology, etc.Other services are Duplex Videoconference between KMIT'L and Chumporn Witthayakhaed , Research Cooperating project between Thai, China and Japan to develop human resources.
