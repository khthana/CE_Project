บทคัดย่อ

บทความนี้เป็นการกล่าวถึงการจดจำตัวอักษรพิมพ์โดยใช้ NN (Neural Network) โดยที่เราจะนำเอา ANN (Artificial Neural Network) มาเป็นตัวที่ใช้จำลองการทำงานของสมองมนุษย์ขึ้นมาอีกที โดยวิธีของ NN นั้นต้องอาศัยลักษณะการผ่านประสบการณ์เรียนรู้ดังเช่นในมนุษย์ แต่ใน NN ที่ประดิษฐ์ขึ้นมาจะยังไม่มีความสามารถที่จะทำงานได้รวดเร็วเทียบเท่ากับการทำงานของมันสมองมนุษย์จริงๆ ซึ่ง NN จะใช้การจดจำลักษณะเด่นของตัวอักษรเป็นอินพุทที่ป้อนให้ระบบ เพื่อนำไปเปรียบเทียบกับตัวอักษรที่เคยผ่านการเรียนรู้ไว้ก่อนแล้ว โดยอาศัยหลักการของวิธี Backpropagation ซึ่งวิธีการดังกล่าวจะใช้การเปรียบเทียบความแตกต่างระหว่างผลลัพธ์ที่ได้กับค่าที่ต้องการ โดยค่าผลต่างที่ได้จะเรียกว่าเป็นค่า Error (ค่าความผิดพลาด) ของระบบ ซึ่งค่า Error นี้จะนำมาใช้ในการปรับแต่งระบบให้ได้ประสิทธิภาพที่ดีขึ้น

Abstract

This paper is about character pattern recognition by using neural network. We use artificial neural network (ANN) to help, as a model of human's brainwork. By the approach of NN, it must be learned through the experience as well as the humans. However, in the NN has not the capability to work as fast as the brain. NN recognize the feature of the character, which apply to the system. To compare with previous trained character by using Backpropagation network approach. This approach uses to compare the differentiation between the actual result and the desired output that called 'Error value' of the system. This error use for improving the performance of the system.
