บทคัดย่อ

ความสามารถของจาวาแอพเพล็ต (Java Applet) และจาวาสคริปต์ (JavaScript) ทำให้เกิดการพัฒนาเว็บไซต์ (Web Site) โดยจะนำไปสู่ลักษณะที่เป็นหนึ่งเดียวกันของเว็บไซต์และนำไปสู่การพัฒนารูปแบบไดนามิค (Dynamic) จาวามีลักษณะที่ดีและมีประสิทธิภาพ ซึ่งพยายามเอาชนะอุปสรรคหลายๆ อย่างในด้านของการไม่รวมกันเป็นหนึ่งเดียว จาวามีลักษณะเป็นพอร์ตอะบิลิตี้ (Portability), ความปลอดภัย (Security), และความทนทาน (Robustness) แต่ในขณะเดียวกันมันเป็นภาษาที่ง่ายและมีประสิทธิภาพสูง และเพราะจาวามีความเป็นออปเจ๊กต์โอเรนเต็ด (Object-Oriented), ไดนามิค, มัลติเทรดดิ้ง (Multithreading) และความทนทาน (robust) มันจึงเหมาะกับอินเตอร์เน็ตและระบบพีซี จาวาสคริปต์มีรูปแบบของคำสั่งคล้ายๆ กับจาวาและมีลักษณะของออบเจ็กต์เบส (Object-based), ซ้อดโค้ดของจาวาสคริปต์จะถูกเขียนรวมอยู่ในแฟ้มข้อมูล HTML ซึ่งหมายความว่าจาวาสคริปต์จะไม่ต้องการการคอมไพล์ แต่จะไปทำการแปลโดยบราวเซอร์ (Browser) ในขณะที่จาวาแอพเพล็ตนั้นจะต้องทำการคอมไพล์ให้เป็นไบต์โค้ด (Byte code) ก่อนที่จะส่งไปให้ผู้ใช้

Abstract

The ability of Java and JavaScript bring to Web site development and unique features to the Web and to the development of dynamic. Java is a powerful program that attempts to overcome the many drawbacks in the way of distributed. Java has attained the portability, security, and robustness necessary, while at the same time keeping the language easy-to-use and at a high-performance level. And because Java is Object-Oriented, Dynamic, multithreading and robust so Java fits in well with the emerging realities of the Internet and personal computing systems. JavaScript has similar syntax as Java and is Object-based. Source code of JavaScript is integrated in the HTML file that means JavaScript must not be compiled but browser interprets the script. Whereas Java Applet must be compile to byte code before sent to user.
