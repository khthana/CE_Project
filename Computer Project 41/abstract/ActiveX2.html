บทคัดย่อ

เอกสารฉบับนี้แบ่งเนื้อหาออกเป็น 2 ส่วน ในส่วนแรกจะเป็นภาพโดยรวมของ ActiveX โดยจะกล่าวถึง ข้อดี - ข้อเสียของ ActiveX เมื่อเปรียบเทียบควบคู่กับ Java , บริการของ ActivX Scripting ที่ทำให้สามารถเพิ่ม Scripting และความสามารถในลักษณะเดียวกับ OLE ให้กับโปรแกรมได้ และ Internet ActiveX Control Pack (ICP) ที่ประกอบด้วย control ชนิดต่าง ๆ ซึ่งเป็นบริการที่สำคัญบนอินเตอร์เนต นอกจากนี้การใช้ ActivX Server Framework จะทำให้เกิดทางเลือกในการ execute application แบบ ISAPI ซึ่งนอกเหนือจากการใช้ CGI ในแบบปกติทั่วไป ในส่วนที่สองจะกล่าวถึงรายละเอียดของ ActiveX ในรูปแบบต่าง ๆ ซึ่งได้แก่ ActiveX Controls , ActiveX Documents และActiveX Components รวมถึงพื้นฐานการเขียนโปรแกรมเพื่อสร้าง application ไว้ใช้งานบน Web page

Abstract

This report composes of two major section. The first section will take you through overview of ActiveX. There are some advantages and disadvantages of ActiveX compared with Java. The ActiveX scripting service empowers you to add scripting and OLE automation capabilities to programs. The Internet ActiveX Control Pack (ICP) contains control for most of major Internet Services. In addition to, ActiveX Server Framework which provides an alternative to CGI method of executing applications on Web server. The second one is sliced into the glory detail of all the major components of ActiveX: ActiveX Controls , ActiveX Documents and ActiveX Components. Including basic concept of programming.
