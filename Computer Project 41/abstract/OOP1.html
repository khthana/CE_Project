บทคัดย่อ

Object - Oriented Design และ Programming เป็นแนวทางและเทคนิคหนึ่งในปัจจุบันที่เลือกใช้ในการออกแบบและเขียนโปรแกรมที่มีขนาดใหญ่และซับซ้อน เพื่อให้สามารถพัฒนาและแก้ไขโปรแกรมต่อไปได้ง่ายขึ้นและมีความเป็นอิสระในการปรับปรุงแก้ไขด้วย ภายในบทความนี้จะกล่าวถึงทฤษฎีซึ่งจะเป็นการนำเสนอแนวทาง และเทคนิคที่ควรจะทราบก่อนการเขียนและออกแบบโปรแกรมประเภท Object รวมถึงมีการกล่าวถึงสัญลักษณ์ต่างๆที่จำเป็นต้องใช้ในการออกแบบเพื่อให้การออกแบบง่ายขึ้น โดยจะมีการยกตัวอย่างโปรแกรมประกอบด้วยเพื่อช่วยให้การอธิบายมีความเข้าใจมากยิ่งขึ้น

Abstract

Object - Oriented Design and Programming is concept and technical of the large and complexity software design and programming option, then it will be easily modify and develop in the future. In this paper presents the advantage and property of object-oriented that needed to know before design and programming, then it have meaning of symbol to be necessary in object-oriented design and have example programming of this design together for easily to understand.
