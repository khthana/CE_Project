บทคัดย่อ

โครงงานนี้มีเนื้อหาที่แบ่งได้ 2 ส่วนใหญ่ๆ ส่วนแรกบรรยายถึง ความหมาย และลักษณะของซุปเปอร์คอมพิวเตอร์ รวมถึงปัญหาต่างๆ ที่ควรใช้ซุปเปอร์คอมพิวเตอร์ในการแก้ปัญหา จากนั้นจะกล่าวถึง ส่วนประกอบภายในของซุปเปอร์คอมพิวเตอร์ และการทำงานของมัน หัวข้อสุดท้ายในส่วนแรกจะกล่าวถึง การคำนวณทางเวคเตอร์ของซุปเปอร์คอมพิวเตอร์ว่าทำงานอย่างไรรวมถึงสถาปัตยกรรมที่ใช้ในซุปเปอร์คอมพิวเตอร์แบบต่างๆ ด้วย ในส่วนที่สองจะเป็นการศึกษาถึง ลักษณะของเครื่องซุปเปอร์คอมพิวเตอร์ที่ใช้ภายในสถาบันว่ามันมีรายละเอียดเป็นอย่างไร

Abstract

This project has two sections: In first section, It explains the meaning and configuration of Supercomputer include the problem that chosen Supercomputer to operate. After that it describes the components and operation of Supercomputer. Finally, it describes the computation of Supercomputer that how it computes including architecture that used in other types of Supercomputer. The second section is studied about Supercomputer (NAWAMAS) in KMIT'L, how about the features and architecture.
