บทคัดย่อ

USB เป็นเทคโนโลยีใหม่ที่เริ่มเข้ามามีบทบาทในวงการคอมพิวเตอร์เป็นอย่างมาก เพราะด้วยประสิทธิภาพที่สูงและความเข้ากันได้ของอุปกรณ์ ในบทความนี้จะกล่าวถึงว่า USB คืออะไร ในปัจจุบันมีเทคโนโลยีใดบ้างที่มีคุณสมบัติใกล้เคียงกัน วัตถุประสงค์ที่ออกแบบ USB ขึ้นมา โครงสร้างทางด้านกายภาพและตรรก การจัดการทางด้านพลังงาน การตรวจสอบความเร็วในการส่งข้อมูล การเชื่อมต่ออุปกรณ์ USB อนาคตของ USB และตัวอย่างของอุปกรณ์ USB ที่อยู่ในท้องตลาดปัจจุบัน ซึ่งจะชี้ให้เห็นแนวทางในการพัฒนาและการนำมาใช้งานในอนาคตได้เป็นอย่างดี
