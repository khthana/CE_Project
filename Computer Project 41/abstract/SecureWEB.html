บทคัดย่อ

ในระบบ Internet ที่เป็นที่นิยมในปัจจุบันนั้น มีระบบรักษาความปลอดภัยที่ใช้กันทั่วไปอย่างแพร่หลายคือ SSL protocol ซึ่งต้องมี transport protocol(TCP) ใช้ในการรับ/ส่งข้อมูล SSL protocol นี้ถือว่าเป็น protocol ที่มีความเป็นอิสระในตัว โดยยึดคุณสมบัติ 3 ประการเพื่อใช้ในการติดต่อ คือ ความเป็นส่วนตัว, มีการรับรองที่ดี, มีความน่าเชื่อถือ ซึ่งยังมีการนำไปใช้เพื่อซื้อขายสินค้าในระบบ internet โดยการใช้ credit card, debit card และ charge card ในการจ่ายเงิน ซึ่งจะต้องมีความปลอดภัยในการซื้อขายด้วยคือ ลูกค้าจะต้องได้รับสินค้าตามที่ ตกลง ส่วนพ่อค้าก็จะต้องได้รับเงินตามที่ตกลงกันไว้

Abstract

The Internet is the most popular system to communicate. To prevent data from hacker it is necessary to use the protocol security system, which one that widespread is SSL protocol. There is transport protocol(TCP) that use for communicate data. It is an independent protocol. Three Purpose of communication, Private, Authenticate, Reliable. Also protect the commerce in WWW that use credit card, debit card, charge card to pay for goods. That means the client should receive the security and product, merchant should get the money in deal.
