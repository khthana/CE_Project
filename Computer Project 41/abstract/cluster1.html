บทคัดย่อ

การประมวลผลแบบกลุ่ม (Cluster) เป็นชนิดของระบบคอมพิวเตอร์แบบขนานที่ได้ทำการพัฒนา ขึ้นเพื่อนำมาทดแทนการใช้เครื่องคอมพิวเตอร์ขนาดใหญ่ในการประมวลผลบางประเภท เช่น การประมวลผลทาง วิทยาศาสตร์ และ วิศวกรรมศาสตร์ ซึ่งจะใช้หลักการนำคอมพิวเตอร์หลายๆเครื่องมาต่อเชื่อมกันด้วยระบบ เน็ตเวิร์คความเร็วสูง แล้วมีการกระจายงาน (Load Balancing) ไปยังเครื่องต่างๆหลังจากประมวลผลแล้วจะนำผลที่ ได้แต่ละส่วนรวมกันเป็นผลลัพธ์สุดท้าย โดยมีโปรแกรมที่ทำหน้าที่ในการจัดการ โดยบทความนี้จะนำเสนอ โปรแกรมที่มีชื่อว่า MPI (Message Passing Interface) ระบบประมวลผลแบบกลุ่มจะมีลักษณะที่แตกต่างจากระบบ การประมวลผลแบบขนาน (Parallel System) และระบบการประมวลผลแบบกระจาย (Distributed System) เช่น ใน การประมวลผลแบบกลุ่มจะสามารถใช้ระบบปฏิบัติการที่แตกต่างกันได้ เป็นต้น การทำระบบประมวลผลแบบ กลุ่มจะต้องใช้ฮาร์ดแวร์และโครงสร้างที่เหมาะสมซึ่งมีให้เลือกหลายรูปแบบ โดยจะมีการยกตัวอย่างผลิตภัณฑ์ที่ม ีการใช้งานจริง คือ Open VMS Cluster ซึ่งเป็นผลิตภัณฑ์ของบริษัท Digital Equipment Corporation (DEC) กล่าวโดยสรุประบบการประมวลผลแบบกลุ่มมีประโยชน์ เช่น ประสิทธิการประมวลผลสูง , ต้นทุนในการทำระบบต่ำ,เพิ่มความสามารถโดยไม่ต้องสร้างขึ้นใหม่และสามารถขยายหรือย่อขนาดของระบบได้

Abstract

Cluster Computing is a type of Parallel Computer System that developed for in place of using Supercomputer to process information such as scientific and mathematics. Cluster brings some computers to link by the high-speed network then the system make Load Balancing to all computers in Cluster. After finished processing then bring packets result of individual computer to make final result. There are programs that manage the function especially. By this article present a content of a program named MPI (Message Passing Interface). Cluster Computing Systems have a different feature from Parallel and Distributed System such as in the Cluster Computing System can use the different operating systems etc. To implement Cluster Computing System, you must use appropriate hardware and structure that have so many choices. There are examples of product that actually implemented such as Open VMS Cluster of Digital Equipment Corporation (DEC) To sum up Cluster Computing System has so many advantages such as high performance, low price/performance, incremental growth and scalability.
