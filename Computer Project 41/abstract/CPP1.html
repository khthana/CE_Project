บทคัดย่อ

การโปรแกรมด้วยภาษา C++ เป็นการเพิ่มความสามารถในการออกแบบโปรแกรมแบบออบเจ็กต์โอเรียนเท็ด (Object Oriented) ลงในการโปรแกรมภาษา C ซึ่งจะช่วยให้โปรแกรมที่สร้างขึ้นสามารถอ่าน เพิ่มเติม ปรับปรุงแก้ไขได้ง่าย มีความยืดหยุ่นและชัดเจน การโปรแกรมด้วยภาษา C++ ถูกออกแบบมาเพื่อให้เกิดความสะดวกในการออกแบบและเขียนโปรแกรม มีการใช้ไลบรารี่ช่วยสนับสนุนการใช้งาน สามารถนำโค้ดที่มีกลับมาใช้ใหม่และทำการปรับปรุงพัฒนาโปรแกรมที่เขียนขึ้นได้ รายงานฉบับนี้ได้นำเสนอถึง วิธีการใช้คีย์เวิร์ด static เพื่อควบคุมการจัดเก็บข้อมูลและการเรียกใช้ข้อมูล รวมถึงการเพิ่มความสามารถในการเข้าถึงข้อมูลผ่านโครงสร้าง Namespace และวิธีการใช้งาน Operator Overloading

Abstract

C++ is an object-oriented extension to the C programming language. It improves productivity by creating at framework in which building an easy -to- read , robust , maintainable , extensible program is the most natural path. There are numerous advantages to using C++. Some of the reasons the language was designed are simplifying the building and use of libraries, allow code re-use and improve maintainability of code.In this report show how static controls strorage and visibility, and an improved way to control access to name via C++'s namespace feature. And how to overload operators and functions
