บทคัดย่อ

ในขบวนการบีบอัดจำเป็นต้องมีองค์ประกอบที่ทำหน้าที่ในการเปลี่ยนรูปข้อมูลจากโดเมนทางเวลาไปเป็นโดเมนทางความถี่ และเปลี่ยนโดเมนทางความถี่กลับไปเป็นโดเมนทางเวลา เทคนิคในการเปลี่ยนรูปมีหลายเทคนิค แต่โดยทั่วไปแล้วเทคนิคการเปลี่ยนรูปที่ใช้ในการบีบอัดข้อมูลที่เป็นรูปภาพกราฟฟิกคือ Discrete Cosine Transform (DCT) ซึ่งเป็นการเปลี่ยนรูปที่ใช้สมการทางคณิตศาสตร์ช่วยในการเปลี่ยนรูป มันจะกำจัด sine coefficient ออกไป ทำให้สมการที่เหลือจะมีเพียงแค่ cosine coefficient เท่านั้น DCT มีทั้งแบบ one-dimension, two-dimension และ three-dimension DCT ซึ่งแบบ three-dimension จะถูกนำไปใช้งานมากกว่าในด้านของภาพ 3 มิติ DCT แตกต่างจากการ transform แบบอื่นตรงที่มันเป็นการเปลี่ยนรูปทางคณิตศาสตร์ ซึ่งเอาท์พุทที่ได้จะไม่มี sine coefficient เลย ปัจจุบัน DCT ถูกนำไปใช้ในเทคโนโลยีหรือเทคนิคต่างๆ มากมาย อาทิเช่น ในเทคโนโลยีการบีบอัดข้อมูล JPEG, การบีบอัดข้อมูลใน Digital Betacam, Inter-code compression เป็นต้น

Abstract

Data compression process need to have component which transform data from time domain to frequency domain and vice versa. There are the varied transformation technique but, generally, technique using in picture or graphic data compression is the discrete cosine transform (DCT). DCT is transformation using mathematical equation to transform with eliminating sine coefficient, so the rest of equation has only cosine coeficient. There are one-dimension, two-dimension and three-dimension DCT. Three-dimension DCT is used to more application than other type , it used for the 3D image. Difference of DCT from other transformation is that being mathematical transformation, its output don't have sine coefficient. Now, DCT is used to apply in a varied technology or technique such as JPEG compression technology, compression in digital betacam and inter-code compression.
