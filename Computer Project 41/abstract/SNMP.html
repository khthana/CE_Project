บทคัดย่อ

ในปัจจุบันเครือข่ายคอมพิวเตอร์มีขนาดใหญ่และมีความซับซ้อนมาก จึงเป็นการยากที่จะดูแลเครือข่ายที่มีลักษณะเช่นนี้โดยไม่ใช้เครื่องมือใดเลย เครื่องมือสำหรับการบริหารเครือข่าย ( Network management Tool ) ที่ถูกพัฒนาขึ้นมาและใช้กันอย่างแพร่หลายบนเครือข่าย TCP/IP ก็คือ ซิมเปิลเน็ทเวิร์กเมเนจเมนท์โปรโตคอล (Simple Network Management Protocol ) หรือ SNMP บทความนี้จะกล่าวถึงซิมเปิลเน็ทเวิร์กเมเนจเมนท์โปรโตคอลโดยการอธิบายแนวคิดพื้นฐานในการบริหารระบบเครือข่าย, ข้อกำหนดของโปรโตคอล SNMP, และยกตัวอย่างของแอพพลิเคชั่นที่นำโปรโตคอล SNMP ไปประยุกต์ใช้งาน

Abstract

Nowadays, Computer networks are very large and complicated. It's very hard to manage such networks without using any tools. The network management tool that was developed and is still in wide use today on TCP/IP network is Simple Network Management Protocol ( SNMP ). This article describes about SNMP by providing basic concepts about network management, protocol specification, and an example of application that use SNMP.
