บทคัดย่อ

ในการเขียนโปรแกรม C++ เราสามารถใช้วิธี Inheritance เพื่อช่วยในการสร้างคลาสใหม่ โดยการใช้ข้อมูลและ ฟังก์ชั่นของคลาสเดิมที่ถูกสร้างขึ้นก่อนหน้านี้ที่มีอยู่แล้ว ซึ่งวิธีนี้ทำให้การเขียนโปรแกรม แบบobject-oriented ทำได้ง่ายขึ้น โดยคลาสเดิมที่มีอยู่แล้วที่นำมาใช้เรียก Base class ส่วนคลาสที่สร้างขึ้นใหม่โดยวิธี Inheritance เรียกว่า Derive class ในการเขียนโปรแกรม C++ นั้นจะมีตัวผู้สร้างแบบสำเนาซึ่งจะถูกใช้ในการสร้างobject ใหม่ โดยจะอ้างอิงถึง object ที่มีอยู่แล้ว ซึ่งเป็นชนิดเดียวกับอาร์กิวเมนต์ของมัน ตัวแปลภาษา C++ จะเรียกตัวผู้สร้างแบบสำเนาโดยอัตโนมัติ เมื่อมีการส่งค่า หรือคืนค่า object แบบส่งผ่านค่า หากผู้เขียนโปรแกรมไม่ได้เขียนตัวผู้สร้างแบบสำเนาไว้ ตัวแปลภาษาก็จะสร้างตัวผู้สร้างแบบสำเนานี้โดยอัตโนมัติ แต่ถ้าผู้เขียนต้องการตัวผู้สร้างแบบสำเนาต่างจากปกติ ก็ต้องเขียนตัวผู้สร้างนี้ขึ้นเอง

Abstract

Inheritance is defined as a means for new class to acquire, access, and use existing methods and data from other classes. It is the feature that enables the true power of object-oriented programming. When a class inherits from another class. Class inherit is called a Base class. A class which inherit from a base class is called a Derived class. The copy-constructor takes a reference to an existing object of the same type as its argument, and it is used to create a new object from an existing one. The compiler automatically calls the copy-constructor when you pass or return an object by value. Although the compiler will automatically create a copy-constructor for you, if you think one will be needed for your class you should always define it yourself to ensure that the proper behavior occurs.
