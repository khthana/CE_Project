บทคัดย่อ

การเขียนโปรแกรมแบบอ้างอิงเชิงวัตถุ (Object Oriented Programming:OOP) เป็นการสร้างส่วนย่อย ๆ ของโปรแกรมที่เรียกว่า Object (โดยแต่ละ Object จะมีคุณสมบัติเฉพาะที่สามารถทำงานบางอย่างได้) ขึ้นมาแล้วนำส่วนย่อยเหล่านั้นมาประกอบกันเป็นโปรแกรมที่ต้องการ โดยที่เราไม่จำเป็นต้องทำการสร้างทุก ๆ ส่วนย่อยของโปรแกรมขึ้นมาใหม่ เราสามารถนำส่วนย่อยที่ต้องการซึ่งได้ถูกสร้างไว้แล้วโดยใครก็ตาม นำมาใช้ได้เลย ภาษาจาวา (Java) ก็เป็นภาษาหนึ่งที่มีลักษณะเป็น OOP ซึ่งจาวามีจุดเด่นมากตรงที่เราทำการเขียนโปรแกรมขึ้นมาเพียงครั้งเดียวแต่สามารถนำโปรแกรมนั้นไปทำการ run ได้ในทุกระบบปฏิบัติการ ดังคำกล่าวที่ว่า " Write once run anywhere" และด้วยเหตุนี้เองที่ทำให้จาวาต้องใช้เวลามากในการ execute โปรแกรม บทความนี้เป็นการอธิบายถึงลักษณะและแนวคิดของการเขียนโปรแกรมในรูปแบบ Object Oriented Programming (OOP) และอธิบายภาษาจาวา (Java) ในระดับเบื้องต้น

Abstract

Object Oriented Programming is the programming technique that separate large program into small pieces which are called "object". Each object will has a unique property that can do something. And we can combine all object to complete main program . We can also reuse the object code for the new program. Java language is an Object Oriented Programming .The main of its advantage is portability and hardware independent as this campaign " Write once run anywhere". But for this benefit, Java spend a lot of time for execution. This paper explain about characteristic and idea of Object Oriented Programming and explain the basic Java language.
