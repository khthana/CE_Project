บทคัดย่อ

โปรแกรมแบบขนานเป็นโปรแกรมที่ถูกพัฒนาสำหรับระบบคอมพิวเตอร์แบบขนาน เช่น Super Computer เพื่อใช้ในการจัดการงานคำนวณที่มีความซับซ้อนและใช้ระยะเวลาในการคำนวณเป็นเวลานาน แนวความคิดในการสร้างโปรแกรมแบบขนานโดยทั่วไปคือ พยายามมองปัญหาให้สามารถแบ่งออกเป็นส่วนๆ ที่ทำงานพร้อมกันได้ในแต่ละตัวประมวลผล โดยจะต้องพิจารณาถึงความแตกต่างทางสถาปัตยกรรมของระบบคอมพิวเตอร์ ที่โปรแกรมนั้นจะถูกทำงานด้วยเพื่อที่จะสามารถดึงเอาขีดความสามารถที่มีอยู่ในระบบคอมพิวเตอร์นั้นมาใช้ให้มีประสิทธิภาพมากที่สุด แนวความคิดและเทคนิคในการพัฒนาโปรแกรมแบบขนานมีดังนี้ Data Parallelism, Relaxed Algorithm, Data Partitioning, Pipelined Computation, Synchronous Iteration, Replicated Worker และ Data Sharing โดยเทคนิคแต่ละแบบจะเหมาะสมกับงาน และสถาปัตยกรรมของระบบคอมพิวเตอร์ที่แตกต่างกัน

Abstract

Parallel Programs are programs that have been developing for parallel computer systems such as Super Computer dealing with sophisticated computations and long-term computing in computations. Main concept, usually, to be applied in writing parallel program is that any works carried out a problem can be divided into pieces equally. A processor together with the others can do each piece of work in the same time. The differences between the architecture of computer systems, that program written to be used, are also the important issue to be considered for getting good performance. Concepts and techniques in developing parallel program are Data Parallelism, Relaxed Algorithm, Data Partitioning, Pipelined Computation, Synchronous Iteration, Replicated Worker and Data Sharing. Each technique is suitable for different architecture of computer systems.
