บทคัดย่อ

โครงงานฉบับนี้นำเสนอเกี่ยวกับระบบเครือข่าย ATM ซึ่งเป็นมาตราฐานใหม่ในการติอต่อสื่อสารที่คิดค้นขึ้นมาเพื่อรองรับการสื่อสารข้อมูลในระบบเครือข่ายต่างๆ ที่มีหลายรูปแบบ โดยจะกล่าวถึงภาพรวมของระบบ ATM, ลักษณะและรูปแบบวิธีการเชื่อมต่อเครือข่าย, การส่งข้อมูลในระบบเครือข่าย ATM, การแบ่งระดับชั้นโดยเปรียบเทียบกับ OSI, การจำลองระบบเครือข่าย LAN, และการรองรับการใช้งานโปรแกรมประยุกต์ต่างๆร่วมกัน ตลอดจนการประยุกต์ใช้งานระบบ ATM เช่นการประชุมทางไกลเป็นต้น

Abstract

In this project, an ATM network system for network communication system, which is new standard for communication is presented. Overview of the system, method of connection, data transfer in ATM network system, comparetion of layer of ATM network system with OSI, LAN emulator and its application (i.e. videoconference etc.) are in this project.
