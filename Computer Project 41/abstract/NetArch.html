บทคัดย่อ

รายงานฉบับนี้ได้กล่าวถึงสถาปัตยกรรมของเครือข่ายคอมพิวเตอร์ซึ่งก็คือการนำส่วนประกอบต่างๆของ เครือข่ายคอมพิวเตอร์มาเชื่อมต่อเข้าด้วยกัน เพื่อทำให้ผู้ใช้คอมพิวเตอร์ในเครือข่ายสามารถติดต่อสื่อสารกันได้ เราสามารถแบ่งเครือข่ายออกได้เป็นหลายรูปแบบโดยอาศัยหลักต่างๆเป็นเกณฑ์ตัดสิน การกำหนดกฎเกณฑ์ในการ ติดต่อสื่อสาร (Protocol)ซึ่งประกอบด้วยหลักในการจัดรูปแบบข้อมูล และวิธีการต่างๆที่ใช้ในการติดต่อสื่อสารก็เป็น ลักษณะหนึ่งของการแบ่งสถาปัตยกรรมเครือข่ายคอมพิวเตอร์ โดยองค์กรมาตรฐานสากล (International Standards Organization: ISO) ได้กำหนดModel มาตรฐานของ Protocol ของเครือข่ายที่เรียกว่า Open System Interconnections (OSI) ไว้ใช้สำหรับการอ้างอิง นอกจากนี้ในรายงานฉบับนี้ยังได้กล่าวถึงรายละเอียดของการทำงานในแต่ละ Layer ของ OSI Reference Model พร้อมทั้งแสดงตัวอย่างสถาปัตยกรรมเครือข่ายคอมพิวเตอร์ในรูปแบบอื่นๆเช่น IBM's System Network Architecture (SNA), DEC's Distributed Network Architecture (DNA), Univac's Distributed Communication Architecture (DCA), Burroughs' Network Architecture (BNA), TRANSDATA (Siemens), Honeywell's Distributed Systems Architecture (DSA), ICL's Information Processing Architecture (IPA) ซึ่งในแต่ละแบบก็จะมีสถาปัตยกรรมและการจัดเรียงชั้น Protocol แตกต่างกันไป จึงได้ทำการเปรียบเทียบความ แตกต่างของแต่ละรูปแบบกับ Reference Model ให้เห็นชัดเจน

Abstract

This report describes Computer Network Architecture, the way to connect network devices to serve communication between users. Computer Network Architecture can be categorized in many ways. Protocol, a formal set of conventions governing the format and control of inputs and outputs between two communicating devices is one of the categorization. International Standards Organization (ISO) has created Open System Interconnections Reference Model as a standard protocol. This report also covers on detail about functions correspond to each layer in OSI Reference Model and review on another example of Computer Network Architectures such as IBM's System Network Architecture (SNA), DEC's Distributed Network Architecture (DNA), Univac's Distributed Communication Architecture (DCA), Burroughs' Network Architecture (BNA), TRANSDATA (Siemens), Honeywell's Distributed Systems Architecture (DSA), ICL's Information Processing Architecture (IPA), which each type has different architecture and model of protocol that can be compared with Reference Model for clearly understand.
