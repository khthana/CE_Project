บทคัดย่อ

ที่อยู่ทางอินเตอร์เน็ตเป็นข้อมูลที่ใช้ระบุถึงเครื่องคอมพิวเตอร์แต่ละตัวที่มีการติดต่อกันในระบบเครือข่ายอินเตอร์เน็ตซึ่งใช้โพรโตคอล TCP/IP เป็นมาตรฐานในการติดต่อ ซี่ง TCP/IP ได้กำหนดให้ใช้ที่อยู่ทางอินเตอร์เน็ตที่เรียกว่า Internet Protocol (IP Address) ในบทความนี้ได้กล่าวถึงรายละเอียดของที่อยู่ทางอินเตอร์เน็ต IPv4 อย่างการแบ่งคลาสของที่อยู่ทางอินเตอร์เน็ต ,การแบ่งเครือข่ายออกเป็นเครือข่ายย่อย,broadcast address และ loopback address เนื่องจากเครือข่ายอินเตอร์เน็ตมีการขยายตัวเพิ่มขึ้นมาก ทำให้จำนวนที่อยู่ทางอินเตอร์เน็ตที่ใช้ในปัจจุบันไม่เพียงพอกับความต้องการ ทำให้เกิดเหตุการณ์ที่เครื่องคอมพิวเตอร์ 2 เครื่องได้รับ IP Address เหมือนกัน บทความนี้จึงได้นำเสนอการแปลงที่อยู่ทางอินเตอร์เน็ตไปเป็นที่อยู่ทางกายภาพโดยใช้ Address Resolution Protocol หรือ ARP ซึ่งใช้เป็นพื้นฐานในการศึกษาผลทางกายภาพที่เกิดขึ้นเมื่อมีการซ้ำกันของที่อยู่ทางอินเตอร์เน็ตและได้นำเสนอแนวทางในการจัดการกับปัญหานี้เพื่อรองรับการขยายตัวของเครือข่ายอินเตอร์เน็ตในอนาคตต่อไป

Abstract

Internet Address uniquely assiged to each host attached to the internet that uses TCP/IP. Internet address in TCP/IP called "Internet Protocol Address (IP Address)". This article will represent about the internet address's detail such as classification of internet address, division of the network into multiple subnetwork, broadcast address and loopback address. Because of internet's expanse is extreme, the current internet address is not enough ,cause the matter that two host has same IP Address. Therefore, this article represent about mapping internet address to physical address (using ARP protocol) for learning about an effect of a scenario when two machine have the same identical IP Address.This article also offered the resolution make ready on internet's expand in the future.
