บทคัดย่อ

Static Model และ Dynamic Model เป็นวิธีการออกแบบอย่างหนึ่งในโปรแกรมเชิงวัตถุ ทั้งสององค์ประกอบจะมีความแตกต่างในแง่ของภาษาC++ คือStatic Modelจะเป็นส่วนการออกแบบคลาส, ตัวแปรสมาชิกและชื่อของฟังก์ชั่นที่ใช้งาน สำหรับDynamic Modelจะเป็นส่วนของการออกแบบInstance ของคลาสต่างๆ กับส่วนInterfaces และส่วนตัวของฟังก์ชั่นสมาชิก Booch's Notation ก็เป็นเครื่องมือที่ช่วยในการออกแบบStatic ModelและDynamic Model โดยStatic Modelจะใช้สัญลักษณ์ต่างๆ ของClass Diagram ส่วนDynamic Modelจะใช้สัญลักษณ์ของObject Diagram การออกแบบนั้น เราต้องสร้างStatic Modelโดยใช้Class Diagramขึ้นมาก่อน จากนั้นนำClass Diagramที่ได้ไปสร้างเป็นObject Diagram ซึ่งจะทำให้เกิดDynamic Model ในส่วนของโปรแกรม C++ จะสร้างขึ้นโดยใช้Static ModelและDynamic Model

Abstract

Static model and Dynamic model are some designing methods in Object Oriented Programming. There are some differences about C++ language in this two components , that is, Static model is a part of designing classes, member variable , and name of using function , whereas Dynamic model is a part of designing instance of classes and interfaces and body of member function. Booch's Notation is also a tool in designing Static model and Dynamic model. Static model usage any sign of classes diagram ,and Dynamic model usage any sign of object diagram. In each designing , we have to draw Static model by using class diagram , First .Then ,we will take that class diagram to draw object diagram, Which will cause Dynamic model to be occur. In part of code program C++ language , we will coding program by using Static model together Dynamic model.
