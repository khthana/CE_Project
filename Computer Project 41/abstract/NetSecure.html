บทคัดย่อ

บทความนี้ได้กล่าวถึงการรักษาความปลอดภัยให้กับระบบเครือข่ายคอมพิวเตอร์ โดยจะเริ่มต้นจากลักษณะของภัยในรูปแบบต่างๆ และการป้องกันระบบเครือข่ายจากภัยอันตรายที่เกิดจากสิ่งแวดล้อมทางกายภาพ ซึ่งในที่นี้จะกล่าวถึงผลกระทบอันเกิดจากแรงเคลื่อนไฟฟ้าไม่คงที่โดยมี UPS เป็นอุปกรณ์ที่ใช้ในการป้องกันระบบจากผลกระทบดังกล่าว และความเสียหายอันเนื่องมาจากภัยธรรมชาติที่ไม่สามารถป้องกันความเสียหายได้ จึงใช้แผนการในการกู้ข้อมูล ที่จะอธิบายถึงแผนการในการกู้ข้อมูลและยุทธวิธีในการ backup ข้อมูลเพื่อลดความเสียหายที่จะเกิดขึ้น การทำรีดันแดนซี่ให้กับการเชื่อมต่อการสื่อสาร วิธีการในการเซฟข้อมูลและสื่อต่างๆที่ใช้ในการ backup ข้อมูล วิธีการจัดการกับระบบ Fault tolerance ซึ่งจะแสดงรายละเอียดของการใช้งานในdiskแบบต่างๆ การป้องกันระบบจากผู้ที่ไม่ได้รับอนุญาตให้เข้าใช้ระบบ โดยจะทำการตรวจสอบผู้ที่จะเข้าใช้ระบบด้วยวิธีการต่างๆ เช่น การใช้ password ตรวจสอบ ภัยที่เกิดจาก software ภายนอกระบบเครือข่าย เช่น ม้าเมืองทรอย ไวรัส และตัวหนอน และท้ายที่สุดจะกล่าวถึง software ที่ใช้ในการป้องกันและต่อต้านภัยอันตรายจาก software ภายนอกระบบ ซึ่งทั้งหมดก็คงจะพอทำให้ผู้อ่านได้เข้าใจถึงปัญหาและวิธีการในการป้องกันและแก้ไขเพื่อเป็นแนวทางในการสร้างความปลอดภัยให้กับระบบเครือข่ายคอมพิวเตอร์ใดๆได้

Abstract

This document describe the security of computer networks, starting with feature of the threats and protecting the network from the harm in physical environment which talk about the effect of the unstable voltage, UPS is the device used to protect the system from this effect and damage from natural disaster that cannot protected so use the disaster recovery plan which describe the process in retrieving data and the strategy in backing-up data to reduce this damage. Producing Redundancy for the communication link. The process in saving data. And media used in backing-up the data. How to manage fault tolerance system, showing the details on how to use different kinds of disks. Protecting the system from unauthorized users by scanning users with different methods such as using password. Danger form software outside the network, e.g. the Trojan horses , virus, and worms. Finally, we will discuss about the software used in defying the danger from software outside the system. Hope this chapter is useful for the readers to understand the problems and their solutions, as an example for establishing security in any computer network.
